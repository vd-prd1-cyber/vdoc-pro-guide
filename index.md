# 서비스 신청하기

V.Doc Pro 도입을 검토 중이시라면, 아래 두 가지 방법으로 문의하실 수 있습니다.

## 서비스 문의하기

### 도입 방법

| 단계                 | 내용                            |
| ------------------ | ----------------------------- |
| STEP 1. 도입 문의 및 상담 | 신청서를 작성하시면 담당자 검토 후 연락드립니다.   |
| STEP 2. 커스텀 문진 설정  | 병원 진료과별 맞춤형 문진 콘텐츠 설정         |
| STEP 3. 시스템 연동     | 병원 EMR에 V.Doc Pro 연동 및 테스트 실행 |
| STEP 4. 교육 실시      | 의료진 대상 사용법 교육 진행 후 정식 운영 개시   |
