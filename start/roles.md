# 역할별 시작 가이드

역할에 맞는 시작 방법을 선택하세요.

## 🩺 의사 (원장님/교수님)

{% hint style="info" %}
처음 3번만 써보면 진료 흐름이 자연스럽게 바뀝니다. **AI 문진 요약 확인**부터 시작해보세요.
{% endhint %}

1. **로그인** — 병원 코드 + 아이디 + 비밀번호로 로그인합니다.
2. **오늘 현황 확인** — 예약 환자 목록과 문진 완료 여부를 한눈에 확인합니다.
3. **AI 문진 요약 확인** — 환자명 클릭 → 증상 요약 탭에서 SOAP 요약을 30초 만에 숙지합니다.
4. **진료 녹음 시작** — 진료 시작 후 **진료 녹음 시작** 버튼을 탭합니다. 대화가 자동 STT 처리됩니다.
5. **진료 저장** — **진료 저장**을 탭합니다. 최종 요약 저장 + 환자 템플릿 자동 전송이 완료됩니다.

---

## 👨‍⚕️ 전공의

{% hint style="info" %}
AI가 생성한 SOAP 초안을 검토·수정하는 방식으로 활용합니다. 직접 입력 시간을 크게 줄일 수 있습니다.
{% endhint %}

1. **환자 목록 접근** — 담당 교수님의 진료과 환자 목록을 확인합니다.
2. **AI 초안 검토** — 증상 요약 탭의 SOAP 초안을 확인하고 필요시 수정합니다.
3. **AI 추천 의증 참고** — MOCA가 제시하는 감별 진단 목록을 임상 추론에 활용합니다.
4. **녹음 요약 확인** — 진료 후 녹음 요약 탭에서 STT 결과를 확인하고 최종 노트를 완성합니다.

---

## 👩‍⚕️ 간호사·코디네이터

{% hint style="info" %}
환자 접수와 문진 링크 발송을 담당합니다. 관리자 화면에서 모든 환자 흐름을 관리할 수 있습니다.
{% endhint %}

1. **문진 링크 발송** — 예약 환자에게 카카오톡/SMS로 문진 링크를 발송합니다. 내원 하루 전 자동 발송도 가능합니다.
2. **QR 체크인 처리** — 환자 내원 시 QR 코드로 체크인 처리합니다. 문진 완료 여부가 자동 확인됩니다.
3. **오늘 현황 모니터링** — 대기·진료중·완료 상태를 실시간으로 확인합니다.
4. **템플릿 전송 확인** — 진료 완료 후 환자 맞춤 템플릿이 발송되었는지 확인합니다.
