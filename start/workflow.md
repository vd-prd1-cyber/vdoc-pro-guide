# 전체 사용 흐름

환자 내원부터 진료 완료까지, V.Doc Pro가 연결되는 방식을 확인하세요.

## 환자 여정

```
📱 문진 링크 수신  →  ✍️ 집에서 문진  →  📲 QR 체크인  →  🏥 진료  →  📩 안내 수신
(카카오톡/SMS)      (AI 맞춤 질문)    (병원 도착)      (집중 진료)   (맞춤 템플릿)
```

## 의사 여정

```
📋 AI 요약 확인  →  🎙️ 녹음 시작  →  🤖 AI 추천 의증  →  💾 진료 저장  →  📤 템플릿 전송
(30초 숙지)        (버튼 탭)         (감별 진단 참고)    (최종 노트)      (자동 발송)
```

## 간호사·매니저 여정

```
📅 일정 확인  →  📲 링크 발송  →  ✅ 체크인 확인  →  📊 현황 모니터링
(예약 목록)     (자동/수동)      (QR 처리)          (실시간 확인)
```

---

## 상세 흐름

### 진료 전

| 단계 | 담당 | 내용 |
|------|------|------|
| 문진 링크 발송 | 간호사·매니저 | 예약 환자에게 카카오톡/SMS 발송 |
| 사전 문진 작성 | 환자 | 집에서 AI 맞춤 문진 완료 |
| QR 체크인 | 환자 + 간호사 | 병원 도착 후 체크인 처리 |
| AI 요약 확인 | 의사 | 증상 요약 탭에서 SOAP 30초 숙지 |

### 진료 중

| 단계 | 담당 | 내용 |
|------|------|------|
| 진료 녹음 시작 | 의사 | 버튼 탭으로 STT 자동 처리 |
| AI 추천 의증 참고 | 의사 | MOCA 감별 진단 목록 확인 |
| 진료 저장 | 의사 | 최종 노트 저장 |

### 진료 후

| 단계 | 담당 | 내용 |
|------|------|------|
| 템플릿 자동 전송 | 시스템 | 환자 맞춤 안내 카카오톡 발송 |
| 현황 업데이트 | 시스템 | 완료 상태로 자동 변경 |
