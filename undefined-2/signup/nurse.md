# 간호사 / 어시스턴트 계정

간호사·어시스턴트 계정은 환자 접수, 문진 링크 발송, 진료 현황 모니터링을 담당합니다.

---

## 계정 생성 방법

관리자가 아래 순서로 진행합니다.

1. **관리자 페이지 → 계정 관리 → "+ 사용자 추가"** 클릭
2. 이름, 이메일 입력
3. 역할에서 **"간호사/어시스턴트"** 선택
4. **초대 이메일 발송** 클릭
5. 해당 직원이 이메일 링크 클릭 → 비밀번호 설정 → 로그인 완료

---

## 간호사 계정으로 할 수 있는 것

| 기능 | 설명 |
|------|------|
| **오늘 현황 모니터링** | 의사별 진료 현황, 환자 카드 상태 실시간 확인 |
| **문진 링크 발송** | 예약 환자에게 카카오톡/SMS로 사전문진 링크 발송 |
| **QR 접수 처리** | 환자 내원 시 QR 코드 체크인 확인 |
| **환자 카드 관리** | 진료 상태 변경, 접수 취소 처리 |
| **템플릿 발송** | 진료 완료 후 환자 맞춤 템플릿 전송 |
| **환자 목록 조회** | 진료 완료 환자 이력, 접수·문진·템플릿 발송 여부 확인 |

---

## 업무 흐름 요약

```
예약 확인 → 문진 링크 발송 → QR 접수 확인 → 오늘 현황 모니터링 → 템플릿 발송 확인
```

{% hint style="info" %}
문진 링크는 예약 확인 문자에 자동으로 포함되어 발송됩니다.\
내원 당일 미완료 환자는 수동으로 재발송하거나, 접수 데스크에서 대신 입력할 수 있습니다.
{% endhint %}

---

## 환자 문진 미완료 시 대처 방법

| 상황 | 대처 방법 |
|------|---------|
| 스마트폰 없는 환자 | 접수 데스크 태블릿에서 직원이 대신 문진 입력 |
| QR 코드 만료 | 오늘 현황 화면에서 QR 재발송 |
| 문진 도중 중단 | QR 재스캔 후 이어서 작성 가능 |
