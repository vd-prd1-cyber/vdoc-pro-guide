# 대표 관리자 (원장님)

관리자 계정은 V.Doc Pro의 모든 설정을 관리하는 최상위 권한입니다.

---

## 최초 관리자 계정 생성

도입 시 aitrics 담당자가 병원 대표 관리자 계정을 직접 생성합니다.\
초대 이메일이 발송되면 링크를 클릭하여 비밀번호를 설정하면 계정이 활성화됩니다.

---

## 관리자 계정으로 할 수 있는 것

| 기능 | 설명 |
|------|------|
| **사용자 관리** | 의료진·간호사 계정 추가, 수정, 잠금, 비밀번호 초기화 |
| **문진 설정** | 진료과별 사전문진 항목 커스터마이징 |
| **템플릿 관리** | 병원 공용 템플릿 생성·수정·비활성화 |
| **대시보드** | 진료 현황 통계 및 문진 완료율 확인 |
| **감사 기록** | 사용자 접속 이력, 설정 변경 이력 조회 |

---

## 추가 관리자 계정 생성

복수의 관리자가 필요한 경우 (예: 원장님 + IT 담당자):

1. **관리자 페이지 → 계정 관리 → "+ 사용자 추가"** 클릭
2. 이름, 이메일 입력 → 역할에서 **"관리자"** 선택
3. **초대 이메일 발송** 클릭

---

## 보안 권고 사항

{% hint style="warning" %}
관리자 계정은 2단계 인증(2FA) 설정을 강력히 권장합니다.\
**관리자 페이지 → 내 계정 → 보안 설정**에서 활성화할 수 있습니다.
{% endhint %}

- 관리자 계정은 필요한 인원만 최소한으로 유지
- 담당자 변경 시 즉시 계정 잠금 또는 권한 변경
- 비밀번호는 정기적으로 변경 권장
