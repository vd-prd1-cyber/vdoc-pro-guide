# 회원가입하기

V.Doc Pro는 별도 설치 없이 직접 가입하고 바로 사용하실 수 있습니다. 가입 후에는 결제 정보 등록 없이 30일 무료 체험이 자동으로 시작됩니다.

### 1. 가입 방법



#### 이메일 가입

이메일 입력 및 인증코드가 포함된 메일이 발송됩니다. 메일이 보이지 않으면 스팸함을 확인하시거나, 60초 후 재발송 버튼을 눌러주세요.

#### 카카오 간편 가입

첫 화면에서 카카오로 간편 가입 버튼을 눌러주세요.

카카오 로그인 화면에서 계정 정보를 입력하고 동의하시면, 이름·이메일·전화번호가 자동으로 등록되어 별도 입력 없이 가입이 완료됩니다.

### 2. 가입 정보 입력

본인 확인을 위한 정보입니다.

### 3. 면허 인증

의료인 면허를 인증하여 전체 기능을 활성화합니다.

도움이 필요하시면 고객센터로 문의해 주세요.
