# 회원가입하기

V.Doc Pro는 별도 설치 없이 누구나 직접 가입하고 바로 사용할 수 있습니다.\
이메일 가입과 카카오 간편 가입 중 편한 방법을 선택하세요.

![](<../.gitbook/assets/screen-signup.png>)

---

## 이메일로 가입하기

### 1단계 — 이메일 입력 및 인증코드 발송

사용할 이메일 주소를 입력하고 **인증코드 발송** 버튼을 클릭합니다.

{% hint style="info" %}
인증코드가 포함된 메일이 발송됩니다. 메일이 보이지 않으면 스팸함을 확인하거나, 60초 후 **재발송** 버튼을 눌러주세요.
{% endhint %}

---

### 2단계 — 인증코드 입력

이메일로 받은 **6자리 인증코드**를 입력합니다.

{% hint style="warning" %}
인증코드는 수신 후 **30분** 이내에 입력해야 합니다. 시간이 초과되면 이전 화면으로 돌아가 코드를 재발송해 주세요.
{% endhint %}

---

### 3단계 — 약관 동의

서비스 이용에 필요한 약관에 동의합니다.

| 구분 | 항목 |
|------|------|
| **필수** (3종) | 서비스 이용약관, 개인정보 처리방침, 의료정보 처리 동의 |
| **선택** (1종) | 마케팅 정보 수신 동의 |

---

### 4단계 — 가입 정보 입력

로그인에 사용할 아이디와 기본 정보를 입력합니다.

| 항목 | 설명 |
|------|------|
| 아이디 | 로그인 시 사용할 아이디 |
| 비밀번호 | 영문·숫자·특수문자 조합 권장 |
| 이름 | 실명 입력 |
| 전화번호 | 고객 지원 및 인증에 사용 |

---

### 5단계 — 가입 완료

가입이 완료되면 환영 화면이 표시됩니다.\
이어서 면허 인증, 병원 정보 입력, 또는 바로 시작하기 중 하나를 선택할 수 있습니다.

---

## 카카오로 간편 가입하기

첫 화면에서 **카카오로 간편 가입** 버튼을 클릭합니다.

카카오 로그인 화면에서 계정 정보를 입력하고 동의하면, 이름·이메일·전화번호가 자동으로 등록되어 별도 입력 없이 가입이 완료됩니다.

{% hint style="info" %}
이미 V.Doc 계정이 연결된 카카오 계정이라면 자동으로 로그인됩니다.
{% endhint %}

---

## 가입 후 다음 단계

가입 완료 후 환영 화면에서 아래 3가지 중 하나를 선택할 수 있습니다.

| 선택 | 설명 |
|------|------|
| **면허 인증하기** | 의료인 면허를 인증하여 전체 기능 활성화 |
| **병원 정보 입력하기** | 소속 병원 정보 등록 |
| **바로 시작하기** | 홈으로 이동 (면허 인증·병원 정보는 나중에 설정 가능) |

---

### 면허 인증하기

면허 인증 방식은 **자동 인증**과 **서류 인증** 중 선택할 수 있습니다.

**자동 인증 (권장)**\
이름·생년월일·전화번호를 입력하면 약 **15초 이내**에 결과를 확인할 수 있습니다.

| 상황 | 안내 |
|------|------|
| 인증 성공 | 면허 인증 완료, 전체 기능 활성화 |
| 정보 불일치로 반려 | 서류 인증으로 전환하여 진행 |
| 일시적 오류 | 잠시 후 재시도하거나 서류 인증으로 진행 |
| 이미 등록된 면허번호 | 고객센터로 문의 |

**서류 인증**\
면허증 이미지를 업로드하면 담당자 검토 후 **영업일 1~2일 이내** 결과를 이메일로 안내드립니다.

{% hint style="warning" %}
이미지가 흐리거나 잘린 경우 반려될 수 있습니다. 면허증 앞·뒤면을 선명하게 촬영하여 업로드해 주세요.
{% endhint %}

---

### 병원 정보 입력하기

소속 병원의 기본 정보를 등록합니다. 가입 후 언제든지 **내 계정 → 병원 정보**에서 수정할 수 있습니다.

| 항목 | 설명 |
|------|------|
| 병원명 | 정식 기관명 |
| 진료과 | 주요 진료과 선택 |
| 병원 이미지 | 프로필용 이미지 (선택) |

---

도움이 필요하시면 [고객센터](../help/contact.md)로 문의해 주세요.
