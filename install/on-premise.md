# On-premise 설치

{% hint style="info" %}
On-premise 방식은 병원 내부 서버에 V.Doc Pro를 직접 설치합니다. 폐쇄망 환경이나 보안 요구사항이 높은 상급 병원에 적합합니다.
{% endhint %}

## 설치 절차

### STEP 1. 인프라 사전 검토

병원 IT 담당자와 AITRICS 엔지니어가 함께 진행합니다.

- 병원 내부 네트워크 환경 확인
- 서버 사양 및 가용 스토리지 검토
- 방화벽·보안 정책 검토
- EMR 연동 방식 결정

### STEP 2. 계약

On-premise 라이선스 계약을 체결합니다.\
연간 유지보수 계약을 포함하여 진행됩니다.

### STEP 3. 서버 설치

AITRICS 엔지니어가 직접 방문하여 설치를 진행합니다.

- 서버 환경 구성 및 V.Doc Pro 설치
- 내부 네트워크 기반 EMR 연동 설정
- 보안 설정 (IP 허용 목록, 접근 제어 등)
- 설치 후 전체 기능 테스트

### STEP 4. 테스트 및 교육

Cloud 방식과 동일하게 AITRICS CES가 역할별 교육을 진행합니다.

---

## 서버 요구사항

병원 규모와 동시 접속자 수에 따라 최적 사양이 달라집니다.\
도입 상담 시 AITRICS 엔지니어가 맞춤 사양을 안내드립니다.

| 항목 | 내용 |
|------|------|
| **운영체제** | Linux (Ubuntu 20.04 LTS 이상) |
| **CPU / 메모리** | 병원 규모에 따라 협의 |
| **스토리지** | 녹음 파일 보관 기간에 따라 협의 |
| **네트워크** | 내부망 전용 또는 DMZ 구성 |

---

## On-premise 운영 유의사항

{% hint style="warning" %}
On-premise 환경에서는 **업데이트 적용이 수동**으로 진행됩니다. 새 기능·보안 패치는 AITRICS 담당자와 일정을 조율하여 적용합니다.
{% endhint %}

- 서버 장애 시 병원 IT 담당자와 AITRICS 기술지원팀이 공동 대응
- 긴급 지원은 도입 시 협의된 담당 CES를 통해 연락
