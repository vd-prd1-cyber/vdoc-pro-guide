# 로그인

{% hint style="info" %}
V.Doc Pro 콘솔은 **Chrome 브라우저에 최적화**되어 있습니다. Chrome 최신 버전을 사용해 주세요.
{% endhint %}

## 로그인 방법

1. Chrome 브라우저에서 V.Doc Pro 접속 URL 입력
2. **병원 코드(기관 코드)** 입력
3. **아이디(이메일)** 및 **비밀번호** 입력
4. **로그인** 클릭

병원 코드는 도입 시 AITRICS 담당자가 별도 안내드립니다.

{% hint style="info" %}
도입 문의는 [문의하기](../help/contact.md)를 이용해 주세요.
{% endhint %}

---

## 로그인 후 역할별 진입 화면

| 역할 | 로그인 후 진입 화면 |
|------|----------------|
| **의료진** | 진료 화면 (오늘 환자 목록) |
| **간호사/어시스턴트** | 오늘 현황 화면 |
| **관리자** | 관리자 설정 화면 |

---

## 비밀번호를 잊으셨나요?

로그인 화면에서 **"비밀번호 찾기"** 클릭\
→ 등록된 이메일로 재설정 링크 발송\
→ 링크 클릭 후 새 비밀번호 설정

비밀번호 재설정이 안 될 경우 소속 병원 관리자에게 요청하거나, AITRICS 기술지원팀에 문의하세요.

{% hint style="info" %}
도입 문의는 [문의하기](../help/contact.md)를 이용해 주세요.
{% endhint %}

---

## 자동 로그아웃 (세션 만료)

보안을 위해 일정 시간 미사용 시 자동 로그아웃됩니다.\
세션 유지 시간은 관리자가 **관리자 페이지 → 보안 설정**에서 변경할 수 있습니다.

| 옵션 | 설명 |
|------|------|
| 30분 / 1시간 / 4시간 / 8시간 | 마지막 활동 후 해당 시간 경과 시 로그아웃 |
| 없음 | 브라우저를 닫을 때까지 유지 |

{% hint style="warning" %}
공용 PC 사용 시 **세션 만료 시간을 짧게** 설정하거나, 사용 후 반드시 로그아웃하세요.
{% endhint %}
