# 환자 관리

진료실 현황을 일단위·주단위로 확인하고, 환자 카드 상태를 직접 관리합니다.

---

## 일단위 보기

| 항목 | 설명 |
|------|------|
| **의사별 진료 현황** | 교수님별 당일 진료 현황을 한 화면에서 확인 |
| **담당 의사 정보** | 현재 진료 중인 의사 이름·진료과·날짜 표시 |
| **환자 카드 이동** | 원활한 환자 관리를 위해 간호사가 진료 상태를 직접 변경 가능 |
| **첨부 서류 확인** | 환자가 업로드한 첨부 서류 목록 확인 가능 |

### 환자 카드 상태 직접 변경

간호사가 환자 카드를 드래그하거나 상태 버튼을 눌러 진료 흐름을 조정할 수 있습니다.

- 대기 중 → 진료 중으로 변경
- 진료 보류 상태 환자를 다시 대기로 복구

---

## 주단위 보기

| 항목 | 설명 |
|------|------|
| **달력 뷰** | 선택한 날짜 기준 주단위 일정 표시 |
| **시간대별 환자 현황** | 1시간 간격으로 환자 진료 현황 확인 |

---

## 문진 링크 발송

| 방법 | 설명 |
|------|------|
| **자동 발송** | 예약 확인 문자에 사전문진 링크 자동 포함 |
| **수동 발송** | 환자 카드에서 수동으로 문진 링크 카카오톡/SMS 발송 |
| **QR 현장 안내** | 접수 데스크 QR 코드로 현장에서 바로 체크인 |

---

## 문진 미완료 환자 대응

| 상황 | 대처 방법 |
|------|---------|
| 링크 문진을 못한 환자 | 접수처 QR 스캔 후 현장에서 문진 시작 |
| 문진 중간에 중단한 환자 | QR 재스캔 후 이어서 작성 또는 처음부터 재시작 |
| 스마트폰이 없는 환자 | 접수 데스크 태블릿으로 직원이 대신 입력 |

{% hint style="warning" %}
**유의사항**\
문진을 진행하지 않은 경우와 문진 중 중단한 경우, QR 접속 후 화면이 다르게 표시됩니다.\
중단한 경우 재시작 또는 이어서 작성 중 선택하도록 안내해 주세요.
{% endhint %}
