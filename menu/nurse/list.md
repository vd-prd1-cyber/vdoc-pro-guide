## 환자 정보 마스킹

환자 목록 우측 상단의 **마스킹 아이콘**을 클릭하여 ON/OFF 전환

* **마스킹 ON** — 환자 이름, 연락처 등 개인정보 일부가 `*`로 표시
* **마스킹 OFF** — 전체 정보 표시

{% hint style="info" %}
공용 모니터나 외부에서 화면이 노출될 수 있는 환경에서는 마스킹 ON을 권장합니다.
{% endhint %}

## 미발송 템플릿 발송

환자 목록에서 **템플릿 미발송** 상태인 환자를 찾아 이후에 발송할 수 있습니다.

1. 해당 환자 행에서 **"템플릿 발송"** 버튼 클릭
2. 적합한 템플릿 선택
3. 발송 확인

과거 날짜의 환자도 발송 가능합니다.
