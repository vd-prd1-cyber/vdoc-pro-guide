# 미니 모드

미니 모드는 V.Doc Pro를 작은 창으로 띄워 EMR 등 다른 프로그램과 함께 사용할 수 있는 방식입니다.\
진료 환경에 맞게 전체 모드와 미니 모드를 유연하게 전환할 수 있습니다.

***

## 미니 모드 전환 방법

진료 화면 우측 상단의 **"미니 모드"** 버튼 클릭\
→ 창이 작은 사이즈로 축소되며 화면 위에 항상 표시됨

전체 모드로 돌아오려면 미니 모드 창의 **"전체 모드"** 버튼 클릭

***

## 미니 모드 화면 구성

| 영역           | 내용                          |
| ------------ | --------------------------- |
| **환자 정보**    | 현재 진료 중인 환자 이름, 문진 요약 간략 표시 |
| **녹음 버튼**    | 진료 녹음 시작 / 중지               |
| **AI 추천 의증** | 문진 기반 감별 진단 목록 확인           |
| **진료 저장**    | 저장 및 템플릿 발송                 |

***

## 미니 모드 활용 팁

{% hint style="success" %}
**추천 사용법**\
EMR 화면을 왼쪽에, V.Doc Pro 미니 모드를 오른쪽 하단에 배치하면\
문진 요약을 보면서 EMR에 바로 복사·붙여넣기할 수 있어 효율적입니다.
{% endhint %}

* 문진 요약의 **원클릭 복사** 기능을 활용하면 EMR 입력 시간을 크게 단축할 수 있습니다
* 진료 중 미니 모드에서도 **녹음 STT**와 **AI 추천 의증** 기능을 그대로 사용할 수 있습니다
* 상세 차팅 편집이 필요하면 전체 모드로 전환하세요
