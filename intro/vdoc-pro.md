# 브이닥 프로 알아보기

## V.Doc Pro란?

V.Doc Pro는 **의료진의 진료를 직접적으로 보조하는 진료실 AI 솔루션**입니다.\
사전문진부터 차팅, 환자 케어 가이드 발송까지, 진료 전 과정에서 반복적인 업무 부담을 줄이고 **진료의 질과 업무 효율성을 함께 높여드립니다.**

현재 **3차 병원 6개 진료과**에서 사용 중입니다.

---

## 진료 전주기 자동화

| 진료 전 | 진료 중 | 진료 후 |
|--------|--------|--------|
| 환자 사전문진 | AI STT 녹음 & 차팅 요약 | AI 환자 케어 가이드 발송 |
| 문진 요약 확인 | AI 추천 의증 | — |

## 주요 기능

### 🔹 문진 Agent — 진료 전, 환자 정보를 미리 파악합니다

환자가 병원 방문 전 스마트폰으로 사전문진을 완료하면, 의료진은 진료 시작 전 AI가 요약한 문진 요약(증상, 복용약, 과거력 등)을 한눈에 확인할 수 있습니다.

반복적인 기초 문진 부담을 줄이고, 의료진이 심화 진료에 집중할 수 있도록 돕습니다.

- **11개 진료과 맞춤 문진 지원** — 내과, 정형외과, 이비인후과, 소아청소년과, 통증의학과, 산부인과, 비뇨기과, 신경과, 재활의학과, 대장항문외과, 가정의학과
- 5만 건의 검증된 의료 데이터를 학습한 자체 AI 모델 적용
- EMR 형식에 맞춰 주증상·병력을 자동 요약, 원클릭 복사·붙여넣기 가능
- 예약 확인 문자에 사전문진 링크 포함 → 환자가 내원 전 집에서 편리하게 작성
- 첨부 서류 OCR 자동 인식으로 타이핑 없이 복사·붙여넣기 가능
- 사전문진 완료 시 PC 메신저 알림 제공

### 🔹 녹음 요약 Agent — 진료 중, 기록 부담 없이 진료에 집중합니다

의료 용어에 최적화된 STT 음성 인식 기술로 진료실 대화를 실시간으로 기록하고, AI가 진료 내용과 진료 계획을 차팅 형식으로 자동 요약합니다.

- 글로벌 기업 모델 대비 최대 **5.8배 빠른 STT 성능** (국내외 7개 모델 비교 기준)
- 사전문진 내용과 진료 녹음을 통합 분석해 최종 요약본 제공
- 의료진 개인의 차팅 스타일을 학습한 맞춤 형식 지원
- 문진 요약을 바탕으로 **AI 추천 의증** 5개 제시 — 의료진이 근거를 확인하고 선택하거나 직접 추가 가능
- 전체 모드 / 미니 모드로 진료 환경에 맞게 유연하게 사용 가능

### 🔹 환자 관리 및 병원 CRM — 진료 후, 환자 소통까지 자동화합니다

진료 완료 후 의료진이 선택한 템플릿을 환자에게 카카오톡으로 자동 발송합니다. 복약 안내, 주의사항, 상비약 안내 등 맞춤형 콘텐츠로 환자 만족도를 높입니다.

- 진료과별 **1,000여 개 추천 템플릿** 기본 제공
- 병원 고유의 맞춤 템플릿 직접 제작 가능
- 오늘 현황 화면에서 환자별 접수·문진·발송 상태를 한눈에 확인
- 일단위·주단위 진료 현황 관리 및 환자 목록 조회 지원

## 시스템 구성

V.Doc Pro는 다음 5가지 구성 요소로 이루어집니다.

| 구성 요소 | 설명 |
|---------|------|
| **환자용 웹 문진** | 별도 앱 설치 없이 링크·QR로 접속하는 모바일/PC 웹 사전문진 화면 |
| **의료진용 PC 프로그램** | 진료 전·중·후 AI 기능과 운영 관리를 통합 제공하는 Windows 프로그램 |
| **MOCA AI 엔진** | 사전문진 분석, 문진 요약 생성, STT 기반 진료 내용 구조화를 수행하는 자체 개발 AI 엔진 |
| **V.Doc DB** | 문진 데이터·진료 기록을 저장하는 데이터베이스 |
| **V.Doc File Storage** | 진료 녹음 원본 및 첨부 서류를 저장하는 파일 스토리지 |

클라우드(Cloud) 환경과 온프레미스(On-Premise) 환경 모두에서 사용 가능합니다.

## 신뢰할 수 있는 AI 기술

V.Doc Pro는 에이아이트릭스의 자체 의료특화 AI 모델 **MOCA(Medical Orchestration Cooperative Agents)**를 기반으로 동작합니다.

| 인증·성과 | 내용 |
|---------|------|
| **글로벌 학회 논문** | EMNLP 2편, NeurIPS 1편, EACL 1편, ICASSP 2편 등 총 6편 게재 |
| **특허** | MOCA 관련 특허 3건 출원 |
| **기술 등급** | TCB 기술평가 **TI-1** 등급 획득 |
| **보안 인증** | ISO 27001 (정보보안 관리체계), ISO 27701 (개인정보보호 관리체계) 인증 획득 |
| **임상 검증** | 신촌세브란스 전공의 40명 참여 교차 검증, 5만 건 고품질 진료 데이터 기반 |

| 단계 | 내용 |
|------|------|
| STEP 1. 도입 문의 및 상담 | 신청서를 작성하시면 담당자 검토 후 연락드립니다. |
