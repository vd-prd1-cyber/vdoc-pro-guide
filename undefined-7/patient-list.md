# 환자 목록 조회하기

내원 환자 목록을 조건별로 조회하고, 선택한 환자의 기본 정보 및 진료 처리 이력을 우측 타임라인으로 빠르게 확인할 수 있습니다.

![](<../.gitbook/assets/screen-patient-list.png>)

---

## 화면 구성

| 영역 | 설명 |
|------|------|
| **상단** | 환자 이름 검색, 담당 의사 필터, 환자 정보 마스킹 토글 |
| **좌측 메인** | 환자 목록 테이블 |
| **우측 사이드** | 선택 환자의 진료 기록 타임라인 |

---

## 검색 및 필터

**환자 이름 검색**\
검색창에 환자 이름 일부 또는 전체를 입력하면 실시간으로 목록이 필터링됩니다.\
검색 결과가 없는 경우 테이블 영역에 noData 상태가 표시됩니다.

**담당 의사 필터**\
담당 의사 드롭다운을 클릭하면 의사 목록이 표시되며, 선택한 의사의 담당 환자만 테이블에 노출됩니다.\
검색 조건과 함께 복합 적용이 가능하며, 기본값은 **담당 의사 (전체)** 입니다.

**환자 정보 마스킹**\
우측 상단 토글로 환자 개인정보 마스킹 여부를 설정할 수 있습니다.

| 상태 | 내용 |
|------|------|
| **ON** | 환자 이름, 생년월일, 나이, 성별 마스킹 처리됨 |
| **OFF** | 권한 범위 내 실제 정보 노출 |

---

## 환자 목록 테이블

테이블 헤더는 스크롤 시 상단에 고정되며, 행 클릭 시 우측 진료 기록 패널이 해당 환자 기준으로 갱신됩니다.\
선택된 행은 강조 배경으로 표시됩니다.

**컬럼 구성**

| 컬럼 | 내용 |
|------|------|
| 일시 | 접수 기준 일시 (YYYY-MM-DD HH:mm) |
| 이름 | 환자명 |
| 생년월일 | 환자 생년월일 |
| 나이 / 성별 | 환자 나이 및 성별 |
| 차트 | 클릭 시 환자 차트 또는 진료 기록 화면으로 이동 |
| 담당 의사 (진료과) | 담당 의사 및 진료과 (상단 필터와 연동) |
| 접수 / 문진 / 템플릿 발송 | 완료 여부를 체크 아이콘으로 표시, 미완료 시 비활성 아이콘으로 표시 |
| 방문 | 누적 방문 횟수 |

---

## 진료 기록 타임라인

환자 행을 클릭하면 우측 패널에 해당 환자의 진료 기록 이력이 시간순으로 표시됩니다.

- 상단 카드: 환자 요약 정보 (이름, 생년월일, 나이/성별)
- 하단 타임라인: 접수, 문진 완료, 진료 보류 등 상태 이력을 시간순으로 표시

{% hint style="info" %}
선택 환자를 변경하면 우측 타임라인 패널 전체가 즉시 갱신됩니다.
{% endhint %}
