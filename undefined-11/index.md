# 대시보드 (통계)

{% hint style="info" %}
대시보드는 **관리자 계정**에서 확인할 수 있습니다.
{% endhint %}

대시보드에서 병원 진료 현황과 V.Doc Pro 활용 지표를 한눈에 확인합니다.

---

## 주요 지표

| 지표 | 내용 |
|------|------|
| **일별 진료 현황** | 예약 수, 접수 수, 진료 완료 수 |
| **문진 완료율** | 전체 환자 중 사전문진 완료 비율 |
| **평균 문진 소요 시간** | 환자가 문진 작성에 걸리는 평균 시간 |
| **템플릿 발송률** | 진료 완료 환자 중 템플릿 발송 비율 |
| **템플릿 사용 현황** | 가장 많이 사용된 템플릿 순위 |
| **진료과별 통계** | 진료과별 환자 수 및 문진 완료율 |

---

## 기간 설정

대시보드 우측 상단에서 조회 기간을 선택합니다.

- 오늘 / 이번 주 / 이번 달 / 직접 입력
- 이전 기간 대비 증감률 표시 (▲▼)

---

## 문진 완료율 높이는 팁

대시보드에서 문진 완료율이 낮게 나타나는 경우 아래를 점검하세요.

| 원인 | 개선 방법 |
|------|---------|
| 문진 링크 발송 시점이 늦음 | 예약 확인 문자에 링크 자동 포함 설정 확인 |
| 문진 항목 수가 많음 | 필수 항목 외 선택 항목 줄이기 |
| 고령 환자 비율 높음 | 접수 데스크 태블릿 활용 안내 강화 |

---

## 데이터 보관

대시보드 통계 데이터는 최대 **1년치**를 보관합니다.\
장기 분석이 필요한 경우 aitrics 담당자에게 데이터 내보내기(Export)를 요청하세요.
