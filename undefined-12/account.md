# 계정 · 권한 관리

## 내 계정 관리

로그인한 사용자의 계정 정보를 확인하고 일부 항목을 수정할 수 있습니다.

![](<../.gitbook/assets/screen-settings.png>)

| 항목 | 수정 가능 여부 |
|------|--------------|
| 가입일 | 수정 불가 |
| 병원 코드 | 수정 불가 |
| 아이디 | 수정 불가 |
| 비밀번호 | **수정하기** 버튼으로 변경 가능 |
| 이름 | 수정 불가 |
| 휴대전화 | **수정하기** 버튼으로 변경 가능 |
| 직무 | 수정 불가 |
| 권한 | 수정 불가 (관리자가 설정) |
| 진료과목 | 수정 불가 |
| 언어 | **변경하기** 버튼으로 변경 가능 |

{% hint style="info" %}
병원 코드와 아이디는 도입 시 발급된 고유값으로 변경할 수 없습니다.\
비밀번호 또는 휴대전화 번호 변경이 필요하면 해당 항목 옆 **수정하기** 버튼을 클릭하세요.
{% endhint %}

## 관리자 설정 · 계정 및 권한 관리

관리자는 소속 병원의 전체 계정을 조회하고, 가입 요청을 승인·반려하거나 계정 사용 여부를 관리할 수 있습니다.

![](<../.gitbook/assets/screen-admin.png>)

**권한 구조**

| 권한 | 주요 기능 |
|------|---------|
| **대표 관리자** | 전체 계정 조회·승인·반려·삭제, 계정 사용 여부 설정, 감사 기록 열람 |
| **일반 관리자** | 진료 화면 이용, 본인 정보 수정 |
| **간호사 · 매니저** | 환자 관리 화면 이용, 본인 정보 수정 |

**가입 요청 승인 · 반려**\
새로운 사용자가 회원가입을 신청하면 관리자에게 알림이 발송됩니다.\
관리자 설정 화면에서 가입 요청 목록을 확인하고 **승인** 또는 **반려**를 처리합니다.

{% hint style="info" %}
승인이 완료되면 해당 사용자에게 승인 완료 알림이 발송되며, 즉시 서비스를 이용할 수 있습니다.\
승인 처리는 일반적으로 영업일 기준 1일 이내 완료됩니다.
{% endhint %}

**계정 사용 여부 설정**\
관리자는 각 계정의 사용 여부를 ON/OFF로 설정할 수 있습니다.\
사용 OFF로 설정된 계정은 로그인이 차단됩니다.
