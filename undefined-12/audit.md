# 감사 기록

{% hint style="info" %}
감사 기록은 **관리자 계정**에서만 조회할 수 있습니다.
{% endhint %}

V.Doc Pro 내에서 발생한 주요 활동 이력을 자동으로 기록합니다.\
보안 감사, 개인정보 처리 현황 확인, 이상 접근 탐지에 활용할 수 있습니다.

![](<../.gitbook/assets/screen-audit-log.png>)

---

## 화면 구성

상단 날짜 네비게이션으로 조회 날짜를 변경할 수 있으며, **다운로드** 버튼으로 해당 날짜의 기록을 내보낼 수 있습니다.

**기록 테이블 컬럼**

| 컬럼 | 설명 |
|------|------|
| 순서 | 기록 순번 |
| 이름 | 활동을 수행한 사용자 이름 |
| 계정 아이디 | 해당 사용자의 로그인 아이디 |
| 권한 | 사용자 권한 (대표관리자, 일반관리자, 매니저 등) |
| 항목 | 활동 분류 (로그인 / 환자 기록 / 계정) |
| 내용 | 활동 상세 내용 |
| 일시 | 활동 발생 일시 |
| IP | 접속 IP 주소 |

---

## 기록 항목 및 내용

**로그인**

| 내용 | 설명 |
|------|------|
| 로그인 성공 | 정상 로그인 |
| 로그인 실패 | 비밀번호 오류 등 로그인 실패 |
| 중복 로그인시 새 위치로 로그인 | 이미 로그인된 계정에서 새 위치로 접속 |

**환자 기록**

| 내용 | 설명 |
|------|------|
| 환자 차트 조회 | 환자 차트 또는 기록 열람 |
| 환자 등록 | 신규 환자 등록 |
| 환자 상태 변경 | 진료 상태 변경 (예: 진료 중) |
| 환자 템플릿 전송 | 환자에게 템플릿 메시지 발송 |

**계정**

| 내용 | 설명 |
|------|------|
| 계정 비밀번호 변경 | 본인 또는 관리자에 의한 비밀번호 변경 |
| 계정 잠금 해제 | 잠긴 계정 해제 처리 |

---

## 다운로드

우측 상단 **다운로드** 버튼을 클릭하면 조회 중인 날짜의 감사 기록을 파일로 내보낼 수 있습니다.\
내보내기 행위 자체도 감사 기록에 자동으로 남습니다.
