# 진료 전 문진 요약 확인

진료 전에 환자가 사전에 응답한 문진 내용을 요약해서 확인할 수 있습니다.\
C.C., Onset, P.I., PHx. 등 핵심 증상 정보를 한눈에 파악하고, EMR에 바로 복사·붙여넣기 할 수 있습니다.

![](<../.gitbook/assets/screen-consultation.png>)

---

## 문진 요약 영역 구성

진료 화면 중앙에 **문진 요약** 섹션이 표시됩니다.

| 노출 요소 | 설명 |
|----------|------|
| **인체 이미지** | 환자가 표시한 증상 부위를 인체 이미지로 시각화 |
| **C.C.** | 주요 호소 증상 |
| **Onset** | 증상 발생 시점 |
| **P.I.** | 현재 증상 상세 (양상, 동반 증상 등) |
| **PHx.** | 과거력 (흡연, 음주, 기저질환 등) |
| **etc.** | 기타 참고 사항 |

---

## EMR 복사 방법

문진 요약 내용을 EMR에 그대로 붙여넣을 수 있도록 **전체 복사**와 **부분 복사**를 지원합니다.

**전체 복사**\
섹션 타이틀 우측 **전체 복사** 버튼을 클릭하면 모든 문진 요약 항목이 한 번에 복사됩니다.

**부분 복사**\
각 항목 옆 복사 아이콘을 클릭하면 해당 항목만 개별 복사됩니다.

{% hint style="info" %}
복사한 내용을 EMR 입력창에 붙여넣기(Ctrl+V)하면 바로 활용할 수 있습니다.
{% endhint %}

---

## AI 문진 전체보기

문진 요약 섹션 우측 상단 **AI 문진 전체보기** 버튼을 클릭하면 환자가 응답한 문진 전체 내용을 상세하게 확인할 수 있습니다.
