# 진료 전 첨부 서류 확인

환자가 사전에 업로드한 첨부 서류를 진료 화면에서 바로 확인할 수 있습니다.\
OCR 기능으로 서류의 텍스트를 자동 인식하여, 직접 타이핑 없이 EMR에 복사·붙여넣기 할 수 있습니다.

![](<../.gitbook/assets/screen-consultation.png>)

---

## 첨부 서류 탭

진료 화면 우측 **환자 첨부 서류 (OCR)** 탭을 클릭하면 환자가 첨부한 서류 목록이 표시됩니다.

| 영역 | 설명 |
|------|------|
| **좌측 썸네일 목록** | 첨부 문서를 세로로 나열, 클릭 시 본문 뷰어에 즉시 표시 |
| **우측 본문 뷰어** | 선택한 문서의 OCR 결과 또는 원본 이미지 확대 뷰 표시 |
| **상단 툴바** | 확대/축소 (100% ~ 500%), 이미지 복사 기능 |

{% hint style="info" %}
선택된 썸네일은 파란색으로 강조 표시됩니다.
{% endhint %}

---

## 복사 방법

**이미지 복사**\
상단 툴바의 **이미지 복사** 버튼을 클릭하면 현재 표시 중인 서류 이미지 전체가 복사됩니다.

**텍스트 복사**\
본문 뷰어에서 원하는 텍스트를 드래그하여 선택한 뒤 복사(Ctrl+C)하면 OCR로 인식된 텍스트만 복사됩니다.

{% hint style="info" %}
복사한 내용을 EMR 입력창에 붙여넣기(Ctrl+V)하면 바로 활용할 수 있습니다.
{% endhint %}
