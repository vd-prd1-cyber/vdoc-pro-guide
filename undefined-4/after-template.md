# 진료 후 · 템플릿 발송하기

{% hint style="warning" %}
템플릿은 진료 후 환자에게 보내는 안내 메시지예요.
{% endhint %}

진료가 끝난 후 환자에게 맞는 안내 템플릿을 선택하여 카카오 메시지로 발송할 수 있습니다.\
1회 최대 **3개**까지 선택하여 동시에 발송할 수 있습니다.

---

## 템플릿 선택 방법

진료 화면 우측 **템플릿** 탭에서 발송할 템플릿을 선택합니다.

**1. 검색으로 찾기**\
상단 검색창에 키워드를 입력하면 제목·카테고리·내용 기준으로 필터링됩니다.

**2. 카테고리로 찾기**\
좌측 카테고리 목록에서 원하는 카테고리를 클릭하면 해당 카테고리의 템플릿만 표시됩니다.\
기본값은 **전체** 상태로 진입합니다.

**3. 즐겨찾기**\
자주 쓰는 템플릿은 ☆ 아이콘을 클릭하여 즐겨찾기에 등록해 두면 빠르게 찾을 수 있습니다.

![](<../.gitbook/assets/screen-template-send1.png>)

---

## 템플릿 미리보기

템플릿 카드를 클릭하면 우측 미리보기 영역에 실제 환자에게 전달되는 메시지 형식으로 내용이 표시됩니다.\
복수 선택 시 미리보기는 **첫 번째 선택 템플릿** 기준으로 표시됩니다.

{% hint style="info" %}
선택된 템플릿 카드는 파란색 테두리로 강조 표시되며, 선택 수는 미리보기 상단에 반영됩니다.
{% endhint %}

---

## 발송 완료 및 미발송

| 상황 | 처리 |
|------|------|
| 템플릿 선택 후 **진료 저장** | 선택한 템플릿이 환자에게 발송됨 |
| 템플릿 선택 없이 **진료 저장** | 미발송 상태로 저장 (추후 발송 가능) |

---

## 환자관리에서 템플릿 발송하기

환자관리 화면에서 진료가 완료된 환자 카드에 있는 **템플릿 발송** 버튼을 눌러 진료 이후에 발송할 수 있습니다.
