# 진료 중 · 녹음 & STT 사용하기

진료 중 대화를 녹음하면 AI가 자동으로 내용을 분석하여 진료 요약을 생성합니다.\
별도 타이핑 없이 진료 내용과 진료 계획을 빠르게 정리할 수 있습니다.

![](<../.gitbook/assets/screen-record.png>)

---

## 녹음 시작하기

진료 화면 하단 **진료녹음** 섹션에서 **시작** 버튼을 클릭하면 녹음이 시작됩니다.

{% hint style="warning" %}
환자가 선택되지 않은 상태에서는 녹음을 시작할 수 없습니다.\
반드시 진료할 환자를 먼저 선택한 후 녹음을 시작해 주세요.
{% endhint %}

---

## 녹음 상태

| 상태 | 설명 |
|------|------|
| **녹음 전** | 시작 버튼 표시 |
| **녹음 중** | 경과 시간 표시, 일시정지 / 취소 / 저장 버튼 표시 |
| **녹음 완료** | 총 녹음 시간 표시, AI 분석 시작 |

{% hint style="info" %}
녹음은 **30초 이상** 진행되어야 저장됩니다.\
녹음 파일은 현재 선택된 환자 및 진료 세션에 자동으로 연결됩니다.
{% endhint %}

---

## 진료 녹음 요약 확인

녹음이 완료되면 AI가 진료 내용을 분석하여 요약을 생성합니다.\
진료 화면 우측에서 **요약** 탭과 **전체대화** 탭으로 확인할 수 있습니다.

| 탭 | 내용 |
|----|------|
| **요약** | 진료내용, 진료계획 등 핵심 항목을 자동 요약 |
| **전체대화** | 녹음된 대화 전체를 텍스트로 확인 |

요약 내용은 **전체복사** 버튼으로 한 번에 복사하거나, 항목별로 개별 복사하여 EMR에 붙여넣기 할 수 있습니다.

---

## 녹음 관리

요약 화면 하단에서 녹음 파일을 관리할 수 있습니다.

| 기능 | 설명 |
|------|------|
| **녹음 삭제** | 현재 진료 세션의 녹음 파일 삭제 |
| **다운로드** | 녹음 파일을 로컬에 저장 |
