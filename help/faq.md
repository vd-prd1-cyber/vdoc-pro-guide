# 자주 묻는 질문 (FAQ)

## 설치·도입

**Q. 도입까지 얼마나 걸리나요?**
계약 후 EMR 연동 환경에 따라 보통 2–4주 소요됩니다. EMR 연동이 필요 없는 스탠드얼론 방식은 1주 이내 설치가 가능합니다.

**Q. 어떤 EMR과 연동되나요?**
현재 연동 가능한 주요 EMR은 도입 상담 시 확인해 드립니다. 연동 목록에 없는 EMR도 API 방식으로 연동 개발이 가능하니 문의 주세요.

**Q. 기존 환자 데이터를 가져올 수 있나요?**
EMR에 저장된 기존 환자 데이터는 EMR 연동을 통해 자동으로 연결됩니다. 별도 마이그레이션 없이 재진 환자 데이터를 바로 활용할 수 있습니다.

## 사용 중 오류

**Q. 문진 QR 코드가 작동하지 않아요.**
다음을 확인해 보세요.
- QR 코드 만료 시간이 지나지 않았는지 확인 (기본 2시간)
- 환자 스마트폰 카메라 권한이 허용되어 있는지 확인
- 관리자 페이지에서 QR을 재발송하거나, 접수 데스크 태블릿으로 대신 문진 입력

**Q. AI 요약이 너무 오래 걸려요.**
AI 요약 생성은 보통 30초 이내에 완료됩니다. 1분 이상 지연되는 경우:
- 브라우저를 새로고침(F5)해 보세요
- 계속 지연되면 support@aitrics.com으로 문의해 주세요

**Q. STT가 제대로 인식되지 않아요.**
- 마이크가 컴퓨터에 정상 연결되어 있는지 확인
- 브라우저 마이크 권한 허용 여부 확인 (브라우저 주소창 자물쇠 아이콘 클릭)
- 주변 소음이 심한 경우 외장 마이크 사용 권장
- 의학 용어 인식이 안 되는 경우 CES 담당자에게 커스텀 사전 추가 요청

**Q. 화면이 로딩되지 않거나 빈 화면이 나와요.**
- 브라우저 캐시 삭제 후 재시도 (Ctrl+Shift+Delete)
- 지원 브라우저(Chrome 최신 버전) 사용 여부 확인
- 네트워크 연결 상태 확인
- 문제가 지속되면 스크린샷과 함께 support@aitrics.com으로 문의

## 데이터·보안

**Q. 환자 데이터는 어디에 저장되나요?**
국내 클라우드 서버(AWS 서울 리전)에 암호화 저장됩니다. 의료정보보호 관련 법령(개인정보보호법, 의료법)을 준수합니다.

**Q. 외부에서 접속이 가능한가요?**
기본적으로 병원 내부 네트워크에서만 접속 가능하도록 설정됩니다. 외부 접속이 필요한 경우(예: 원격 진료) 관리자 설정에서 IP 허용 범위를 조정하거나 VPN 연동을 요청하세요.

**Q. 데이터 백업은 어떻게 되나요?**
매일 자동 백업이 수행되며 30일치 백업이 보관됩니다. 데이터 복구가 필요한 경우 AITRICS 기술 지원팀에 요청하세요.

## 기능

**Q. 여러 진료과가 있는 병원에서 사용할 수 있나요?**
네. 진료과별로 다른 문진 프로파일과 템플릿을 설정할 수 있습니다. 환자 접수 시 선택된 진료과에 따라 적절한 문진이 자동 제공됩니다.

**Q. 태블릿이나 모바일에서도 사용할 수 있나요?**
V.Doc Pro는 웹 기반으로, iPad·Android 태블릿의 최신 Chrome 브라우저에서 사용 가능합니다. 최적 경험을 위해 PC·노트북 환경을 권장합니다.

**Q. 오프라인(인터넷 미연결) 환경에서도 사용할 수 있나요?**
현재는 인터넷 연결이 필요합니다. 원내 폐쇄망 환경의 경우 별도 온프레미스 구축을 문의해 주세요.

## 교육·지원

**Q. 직원 교육은 어떻게 받나요?**
도입 시 AITRICS CES(Customer Engineering Specialist)가 직접 방문하여 역할별 교육을 진행합니다. 교육 후 이 가이드와 영상 자료를 제공드립니다.

**Q. 새로운 직원이 입사하면 어떻게 교육하나요?**
이 가이드의 [역할별 시작 가이드](../start/roles.md)를 참고하시거나, CES 담당자에게 재교육을 요청하실 수 있습니다.
