# 회원가입

V.Doc Pro는 역할에 따라 계정 종류가 다릅니다.\
도입 시 aitrics 담당자가 최초 계정 설정을 함께 진행하며, 이후 추가 계정은 관리자가 직접 생성할 수 있습니다.

---

## 역할 체계

| 역할 | 주요 권한 | 해당 직군 |
|------|---------|---------|
| **관리자** | 모든 설정 변경, 사용자 계정 관리, 통계 조회 | 원장님, IT 담당자 |
| **의료진** | 진료 화면 접근, AI 문진 요약 확인, 녹음 STT, 차팅 | 원장, 교수, 전공의 |
| **간호사/어시스턴트** | 환자 접수, 문진 링크 발송, 오늘 현황 관리 | 간호사, 접수 직원 |

---

## 계정 생성 방법

### 최초 도입 시 (aitrics 담당자 진행)

도입 계약 완료 후 aitrics CES 담당자가 아래를 직접 설정합니다.

1. 병원 코드(기관 코드) 발급
2. 관리자 계정 1개 이상 생성 및 초대 이메일 발송
3. 진료과 구성 및 초기 문진 설정

### 추가 계정 생성 (관리자 직접 진행)

관리자 계정으로 로그인 후 **관리자 페이지 → 계정 관리 → "+ 사용자 추가"** 에서 직접 생성합니다.

→ 자세한 내용은 [계정·권한 관리](../admin/account.md) 참고
