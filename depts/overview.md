# 진료과별 활용 가이드

{% hint style="info" %}
V.Doc Pro는 11개 주요 진료과에 맞춤화된 문진 항목과 템플릿을 기본으로 제공합니다. 각 진료과의 특성에 따라 다른 방식으로 활용할 수 있습니다.
{% endhint %}

## 진료과별 핵심 포인트

### 🫀 내과

| 활용 포인트       | 설명                               |
| ------------ | -------------------------------- |
| **만성 질환 추적** | 고혈압·당뇨·고지혈증 지표 변화를 방문마다 자동 추적    |
| **복약 관리**    | 복용 중인 약물 목록 문진 수집, 상호작용 경고 자동 표시 |
| **재진 비교**    | 이전 방문 증상·소견과 현재 상태 자동 비교         |

**추천 워크플로우**: 문진 → AI 요약(만성 지표 강조) → 진료 → 템플릿(만성질환 추적 양식)

***

### 🦴 정형외과

| 활용 포인트     | 설명                    |
| ---------- | --------------------- |
| **통증 스케일** | 부위별 통증 강도·양상 구조화 수집   |
| **활동 제한**  | 일상·업무·스포츠 활동 제한 정도 문진 |
| **시술 추적**  | 주사·시술 후 경과 추적 템플릿     |

**추천 워크플로우**: 문진(통증 특화) → 진료 중 STT → 템플릿(ROM·통증 기록)

***

### 👶 소아과

| 활용 포인트     | 설명                          |
| ---------- | --------------------------- |
| **성장 지표**  | 키·몸무게 백분위 자동 계산 및 표시        |
| **예방접종**   | 접종 이력 확인 및 미접종 알림           |
| **보호자 소통** | 보호자가 대신 문진 작성, 아이 증상 정확히 전달 |

{% hint style="warning" %}
소아과의 경우 보호자가 문진을 작성합니다. QR 코드를 보호자 스마트폰으로 전달하거나 접수 데스크에서 태블릿으로 작성할 수 있습니다.
{% endhint %}
