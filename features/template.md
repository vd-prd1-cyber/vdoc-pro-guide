# 진료 후 템플릿

{% hint style="info" %}
진료가 끝난 후 EMR에 기록할 내용을 구조화된 템플릿으로 빠르게 완성합니다. AI가 문진 요약과 STT 내용을 자동으로 채워 넣어, 의사는 확인·수정만 하면 됩니다.
{% endhint %}

## 템플릿이 해결하는 문제

진료 후 EMR 기록은 반복적이고 시간이 많이 걸립니다. 특히 같은 질환의 환자가 여럿일 때 유사한 내용을 반복 입력하는 것은 비효율적입니다.

V.Doc Pro 템플릿은 진료과·질환별로 구조화된 양식을 제공하고, AI가 해당 환자의 데이터를 자동으로 채워 의사가 **최소한의 수정**으로 기록을 완성할 수 있게 합니다.

## 템플릿 구성

### SOAP 구조 기본 제공

| 섹션 | 내용 | AI 자동 채움 |
|-----|------|------------|
| **S** (Subjective) | 환자 주 호소, 증상 기술 | ✅ 문진 요약에서 |
| **O** (Objective) | 이학적 검사 소견, 바이탈 | 부분 (의사 직접 입력) |
| **A** (Assessment) | 진단명, 감별 진단 | ✅ AI 요약에서 |
| **P** (Plan) | 처방, 교육, 추적 계획 | ✅ STT에서 |

### 진료과별 특화 템플릿

| 진료과 | 특화 항목 |
|-------|---------|
| 내과 | 만성 질환 지표 추이, 투약 순응도 |
| 외과 | 수술 전후 소견, 창상 상태 |
| 소아과 | 성장 지표, 예방접종 이력 |
| 정형외과 | 관절 ROM, 통증 스케일 |
| 피부과 | 병변 위치·크기, 사진 링크 |
| 산부인과 | 임신 주수, 검사 결과 |

## 사용 방법

### 1. 진료 종료 후 템플릿 열기

진료 화면에서 **"기록 작성"** 버튼을 클릭합니다. 해당 환자의 진료과·증상에 맞는 템플릿이 자동으로 선택됩니다.

### 2. AI 채움 내용 확인

문진 요약과 STT에서 추출된 내용이 각 섹션에 자동으로 채워져 있습니다.

```
[S] 3일 전부터 인후통 및 발열(38.5°C). 연하 시 통증 심해짐.
[O] 편도 발적, 황백색 삼출물 확인. 경부 림프절 압통(+).
[A] 급성 편도염 (J03.9) — 연쇄구균 감염 의심
[P] 아목시실린 500mg 1일 3회 × 5일, 해열진통제 PRN
    3일 후 재진 권고, 격리 및 수분 섭취 교육
```

### 3. 수정 및 저장

필요한 부분을 수정하고 **"EMR에 저장"** 버튼을 클릭합니다. 연동된 EMR 시스템에 즉시 전송됩니다.

{% hint style="success" %}
**평균 기록 시간 67% 단축**
기존 수동 입력 대비 진료 후 기록 소요 시간을 평균 67% 줄입니다.
{% endhint %}

## 나만의 템플릿 만들기

자주 사용하는 문구나 진료 패턴을 저장해 개인 템플릿으로 관리할 수 있습니다.

1. 기록 완성 후 **"템플릿으로 저장"** 클릭
2. 템플릿 이름 입력 (예: "상기도감염 기본")
3. 다음 진료부터 해당 템플릿 불러오기 가능

관리자는 병원 전체 공용 템플릿을 설정할 수 있습니다. [템플릿 관리](../admin/template-management.md)를 참고하세요.
