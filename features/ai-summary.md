# AI 문진 요약

{% hint style="info" %}
MOCA(Medical Orchestration Cooperative Agents)가 문진 데이터와 과거 진료 기록을 분석하여, 진료에 바로 활용할 수 있는 **구조화된 요약**을 자동 생성합니다.
{% endhint %}

## MOCA가 만드는 요약

환자가 문진을 완료하는 순간, AI가 즉시 분석을 시작합니다. 의사가 진료실에서 환자를 호출할 때 이미 요약이 준비되어 있습니다.

### 요약 구성 요소

| 항목 | 설명 |
|-----|------|
| **한줄 인상** | "38.5°C 발열 + 인후통, 편도염 의심" 형태의 핵심 요약 |
| **주요 증상** | 부위·강도·기간을 정리한 증상 리스트 |
| **감별 진단 힌트** | 증상 패턴 기반 감별 질환 제안 |
| **주의 플래그** | 응급 징후, 약물 상호작용, 알레르기 경고 |
| **이전 방문 비교** | 재진 환자의 경우 증상 변화 추이 |

{% hint style="success" %}
**84.6% 진단 정확도**
AITRICS MOCA는 서울대병원·삼성서울병원 등에서 임상 검증을 완료하였으며, 84.6%의 진단 정확도를 기록했습니다.
{% endhint %}

## 실제 화면 흐름

```
[환자 목록] 환자 이름 클릭
    ↓
[문진 요약 패널] 우측에 AI 요약 자동 표시
    ↓
의사가 요약 확인 → 진료 시작
    ↓
필요 시 [전체 문진 보기] 클릭으로 원문 확인
```

## 요약 정확도를 높이는 팁

{% hint style="warning" %}
AI 요약은 환자가 입력한 문진 데이터의 품질에 영향을 받습니다. 환자가 문진을 성실히 작성할수록 요약의 정확도가 높아집니다.
{% endhint %}

- **문항 완성도**: 필수 항목(주 증상, 발생 시점)이 채워진 경우 정확도 향상
- **과거 진료 데이터 연동**: EMR 연동이 활성화된 경우 재진 환자 분석 정밀도 상승
- **진료과 설정**: 진료과에 맞는 문진 템플릿을 사용할 때 관련 감별 진단 정확도 향상

## 요약 결과 활용하기

요약 결과는 진료 중 언제든지 참고할 수 있으며, 진료 후 작성하는 [진료 후 템플릿](./template.md)에 자동으로 불러올 수 있습니다.

요약 내용 중 수정이 필요한 항목은 의사가 직접 편집할 수 있습니다. AI의 제안은 **참고 자료**이며, 최종 진단과 처방은 항상 의사의 판단에 따릅니다.
