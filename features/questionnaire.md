# 사전 문진

{% hint style="info" %}
환자가 진료실에 들어오기 **전**에 증상과 병력을 수집합니다. 의사는 이미 파악된 정보로 진료를 시작할 수 있습니다.
{% endhint %}

## 어떻게 동작하나요?

```
환자 도착 → QR 코드 스캔 또는 키오스크 접수
          → 스마트폰으로 문진 응답 (평균 3분)
          → AI 분석 완료
          → 의사 화면에 요약 표시
          → 진료 시작
```

환자는 대기 중에 스마트폰으로 문진을 완료하고, 의사는 진료실 입장 전 이미 핵심 정보를 확인할 수 있습니다.

## 문진 항목 구성

사전문진은 두 가지 유형의 문항으로 구성됩니다.

| 문항 유형 | 설명 |
|---------|------|
| **커스텀 문항** | 의료진이 진료과·교수님별로 직접 설정·커스터마이징하는 고정 문항 |
| **AI 문항** | 커스텀 문항과 환자 답변을 이해하여 임상적으로 연관된 추가 질문을 MOCA가 자동 생성 |

두 유형이 결합되어 각 진료 환경에 최적화된 문진이 가능하며, MOCA가 환자 답변을 기반으로 증상 흐름·주요 호소를 구조화하여 의료진 화면에 AI 요약으로 전달합니다.

| 카테고리 | 수집 항목 |
|---------|---------|
| **주 증상** | 부위, 강도(1–10), 발생 시점, 양상 |
| **동반 증상** | 발열, 오한, 구역, 두통 등 |
| **과거력** | 만성 질환, 수술 이력 |
| **복용 약물** | 현재 복용 중인 처방·비처방약 |
| **알레르기** | 약물·음식 알레르기 |
| **생활 습관** | 흡연, 음주 (진료과별 설정 가능) |

{% hint style="success" %}
**진료과별 맞춤 문항**
내과·외과·소아과·피부과 등 진료과 특성에 맞게 문항을 커스터마이즈할 수 있습니다. 자세한 내용은 [문진 설정](../admin/questionnaire-settings.md)을 참고하세요.
{% endhint %}

## 성능 및 기술력

{% hint style="success" %}
**증상 분류 정확도 84.6%**
MOCA의 문진 후 증상 분류 정확도는 실제 의료 환경에서 84.6%를 기록했습니다. 이 수치는 신촌세브란스 전공의 40명이 참여한 임상 교차 검증과 5만 건 규모의 고품질 진료 데이터 구축을 기반으로 도출되었으며, 의사 및 GPT 대비 우수한 결과를 입증하였습니다.
{% endhint %}

## 의사 화면에서 보이는 것

진료실 화면에서 환자를 선택하면 문진 요약이 바로 표시됩니다.

- **주 호소**: 환자가 직접 입력한 핵심 증상 (AI 요약 포함)
- **바이탈 플래그**: 고열·혈압 이상 등 이상 소견 강조 표시
- **과거력 요약**: 주요 만성 질환·수술 이력 한눈에 확인
- **복약 목록**: 상호작용 주의 약물 자동 하이라이트

{% hint style="warning" %}
**미응답 환자**
환자가 문진을 완료하지 않은 경우에도 진료는 정상적으로 진행됩니다. 이 경우 의사가 직접 문진을 진행하거나, 진료 중 STT 녹음으로 대체할 수 있습니다.
{% endhint %}

## 자주 묻는 질문

**Q. 환자가 QR 코드를 스캔하기 어려워하면 어떻게 하나요?**
접수 직원이 환자 대신 키오스크에서 문진을 대신 입력할 수 있습니다. 또는 간호사 화면에서 직접 문진 항목을 입력하는 것도 가능합니다.

**Q. 문진 데이터는 EMR에 자동으로 저장되나요?**
네. V.Doc Pro는 연동된 EMR 시스템에 문진 결과를 자동 전송합니다. 저장 형식 및 매핑은 도입 시 설정됩니다.

**Q. 환자 개인정보는 어떻게 보호되나요?**
모든 문진 데이터는 국내 의료정보보호 기준(개인정보보호법, 의료법)을 준수하여 암호화 전송·저장됩니다.
