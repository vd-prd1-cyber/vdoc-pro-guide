# 진료 중 녹음·STT

{% hint style="info" %}
진료 대화를 실시간으로 텍스트로 변환합니다. 의사는 모니터를 보지 않고 환자에게 집중하면서, 대화 내용이 자동으로 기록됩니다.
{% endhint %}

## 핵심 가치

진료 중 EMR에 내용을 타이핑하거나 메모하는 시간을 없앱니다. 녹음 버튼 하나로 대화 전체를 캡처하고, AI가 진료 관련 내용만 추출·정리합니다.

## 사용 방법

### 1. 녹음 시작

진료 화면 상단의 **녹음 버튼(●)** 을 클릭합니다. 빨간 점이 깜박이면 녹음 중입니다.

```
[●  녹음 중]  00:02:34
```

{% hint style="warning" %}
녹음 전 환자에게 녹음 사실을 고지하고 동의를 구하는 것을 권장합니다. V.Doc Pro는 동의서 팝업을 자동으로 표시하는 기능을 제공합니다 (관리자 설정에서 활성화 가능).
{% endhint %}

### 2. 진료 진행

평소와 동일하게 진료를 진행합니다. 화면 우측에 실시간 STT 텍스트가 스크롤됩니다.

- 실시간 텍스트는 **참고용**으로 즉시 확인 가능
- 발화자 구분: 의사 / 환자 자동 분리
- 의학 용어, 약물명, 수치 인식 최적화

### 3. 녹음 종료 및 요약

**중지(■)** 버튼을 누르면 AI가 즉시 핵심 내용을 추출합니다.

| 추출 항목 | 예시 |
|---------|------|
| 주요 호소 | "3일 전부터 목이 아프고 열이 남" |
| 의사 소견 | "편도 발적, 삼출물 확인됨" |
| 처방 계획 | "항생제 5일 처방, 3일 후 재진 권고" |
| 환자 교육 | "수분 섭취 충분히, 격리 권고" |

### 4. 진료 후 템플릿 연동

추출된 내용은 [진료 후 템플릿](./template.md)에 자동으로 불러와져 EMR 기록 작성 시간을 최소화합니다.

## 성능 및 기술력

{% hint style="success" %}
**임상 환경 STT 정확도 91% 이상**
V.Doc의 STT는 임상 환경에서 91% 이상의 음성→텍스트 변환 성능을 확보하였습니다(운영 환경에 따라 변동 가능). 이 수치는 5만 건 규모의 고품질 진료 데이터 구축을 기반으로 도출되었으며, 의사 및 GPT 대비 우수한 결과를 입증하였습니다.
{% endhint %}

사전문진 결과와 STT 텍스트는 MOCA가 함께 분석하여 진료 노트 초안을 생성합니다. 의료진이 검토 후 EMR에 복사·붙여넣기로 반영하며, 최종 입력은 주치의 판단에 따릅니다.

## 언어 및 음질

{% hint style="success" %}
**한국어 의료 특화 STT**
일반 STT 엔진 대비 의학 용어, 약물명, 한국 의료 관용어 인식률이 높습니다. 사투리·빠른 말투도 인식합니다.
{% endhint %}

- **권장 마이크**: 진료 책상 위 단일 지향성 USB 마이크 또는 헤드셋
- **주변 소음**: 조용한 진료실에서 최적 성능 (일반적인 진료실 소음 수준은 정상 작동)
- **인식 언어**: 한국어 (향후 영어 지원 예정)

## 자주 묻는 질문

**Q. 녹음 파일은 어디에 저장되나요?**
녹음 원본 파일은 병원 서버에 암호화 저장됩니다. 저장 기간은 관리자 설정에서 조정할 수 있습니다.

**Q. 마이크가 없어도 사용할 수 있나요?**
노트북 내장 마이크도 사용 가능하지만, 음질 향상을 위해 외장 마이크를 권장합니다. 설치 시 CES 담당자가 최적 마이크를 추천드립니다.

**Q. 환자가 녹음을 거부하면 어떻게 하나요?**
STT 녹음은 선택 기능입니다. 환자 동의가 없는 경우 녹음 없이 다른 기능(문진 요약, 템플릿)만 사용할 수 있습니다.
