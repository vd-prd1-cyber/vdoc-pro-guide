# 환자 관리하기

교수님별 진료실 현황을 일단위 또는 주단위로 확인하고, 환자 카드 상태를 직접 변경할 수 있습니다.

---

## 일단위 보기

![](<../.gitbook/assets/screen-patients-daily.png>)

당일 진료 환자 현황을 **진료 중 · 진료 보류 · 진료 완료** 컬럼으로 구분하여 확인합니다.

**날짜 및 의사 선택**\
좌측 달력에서 날짜를 선택하고, 하단 의사 목록에서 담당 교수님을 선택하면 해당 날짜의 진료 건이 호출됩니다.\
기본값은 오늘 날짜, 해당 병원의 의사 전체 선택 상태입니다.

**일단위 / 주단위 전환**\
상단 **일단위 · 주단위** 탭으로 보기 방식을 전환할 수 있습니다.

**환자 정보 영역**\
좌측 환자 정보 패널에서 선택된 환자의 개인정보와 방문 기록을 확인할 수 있습니다.

| 항목 | 내용 |
|------|------|
| 개인정보 | 이름, 전화번호, 주민등록번호, 성별/나이 |
| 방문 기록 | 접수일시, 초진/재진, 접수 상태, 문진 상태, 템플릿 발송 상태, 환자 차트 |

{% hint style="info" %}
**차트 보기** 버튼을 클릭하면 해당 환자의 차트 화면으로 이동합니다.\
일단위에서는 환자 정보 수정이 불가하며, 주단위에서 [정보변경] 버튼을 통해 수정할 수 있습니다.
{% endhint %}

**환자 카드 이동**\
간호사가 환자 카드의 상태 버튼을 클릭하여 진료 상태를 직접 변경할 수 있습니다.

| 버튼 | 동작 |
|------|------|
| **서류** | 환자 첨부 서류 확인 |
| **진료중** | 진료 중 상태로 이동 |
| **문진완료** | 문진 완료 상태 표시 |
| **템플릿 발송** | 완료된 환자에게 템플릿 발송 |

---

## 주단위 보기

![](<../.gitbook/assets/screen-patients-weekly.png>)

해당 주의 접수 완료 환자 목록을 시간대별로 확인합니다.

**화면 구성**

| 영역 | 설명 |
|------|------|
| 날짜 영역 | 해당 주의 날짜(월~일) 표시 |
| 시간대 영역 | 진료 시간대별 슬롯 표시 |
| 환자 리스트 | 해당 시간대의 접수 환자 목록 |
| 더보기 | 표시 한도 초과 시 추가 환자 목록 확인 |

**환자 리스트 색상 구분**

| 색상 | 의미 |
|------|------|
| 노란색 | 건강 체크 생략 건 |
| 하늘색 | 건강 체크 완료 건 |
