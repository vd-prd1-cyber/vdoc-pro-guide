# 사전 문진 설정

진료과별 사전문진 항목을 확인하고, 개인 설정을 관리합니다.

{% hint style="info" %}
사전문진 항목의 추가·수정·삭제는 **관리자 권한**이 필요합니다.\
자세한 설정은 [문진 설정 (관리자)](../admin/questionnaire-settings.md)를 참고하세요.
{% endhint %}

---

## 사전문진 구성 확인

의료진은 본인 진료과에 설정된 문진 항목을 확인할 수 있습니다.

### 문진 흐름 (환자 화면 기준)

```
본인 확인 (공통)
    ↓
첨부 서류 업로드 / OCR 인식 (공통)
    ↓
주 증상 입력
    ↓
진료과별 커스텀 문진 (11~15개 문항)
    ↓
AI 추가 문진 5개 (자동 생성)
    ↓
문진 제출 완료
```

### 커스텀 문진 답변 타입

| 타입 | 설명 | 예시 |
|------|------|------|
| **단일 선택** | 보기 중 하나만 선택 | 흡연 여부 (예/아니오) |
| **복수 선택** | 보기 중 여러 개 선택 가능 | 동반 증상 (두통, 오한, 발열 등) |
| **숫자 입력** | 숫자 직접 입력 | 통증 강도 (0–10), 체중 |
| **글자 입력** | 자유 텍스트 입력 | 복용 중인 약물명 직접 기재 |

---

## 진료과별 커스텀 문진 예시

| 진료과 | 주요 커스텀 문진 항목 |
|-------|------------------|
| **유방외과** | 과거력, 현재 복용 약물, 임신·출산 이력, 호르몬 노출 기간, 가족력 |
| **이비인후과** | 직업·환경 노출 여부 |
| **내분비내과** | 현재 복용 약물 (당뇨·갑상선 관련) |
| **순환기내과** | 최근 시술·수술 이력 |
| **소화기내과** | 과거력 (내시경 결과 포함) |
