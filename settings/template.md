# 템플릿 설정

자주 사용하는 진료 후 안내 문구를 개인 템플릿으로 저장하고 관리합니다.

---

## 템플릿 사용 방법

### 진료 중 템플릿 선택

1. 진료 화면에서 진료 완료 전 템플릿 탭 클릭
2. 카테고리를 선택하거나 검색으로 원하는 템플릿 찾기
3. 템플릿 선택 → 내용 미리보기 확인
4. 필요 시 해당 환자에게만 적용되는 수정 가능
5. **진료 저장** 클릭 시 선택한 템플릿이 환자에게 자동 발송

{% hint style="info" %}
진료 화면에서 템플릿 내용을 수정하면 **해당 환자에게만** 수정된 내용이 발송됩니다.\
원본 템플릿은 변경되지 않습니다.
{% endhint %}

---

## 개인 템플릿 관리

### 즐겨찾기 설정

자주 사용하는 템플릿을 즐겨찾기로 등록하면 진료 시 빠르게 찾을 수 있습니다.

1. 설정 → 템플릿 목록에서 원하는 템플릿의 **⭐ 즐겨찾기 아이콘** 클릭
2. 진료 화면 템플릿 탭에서 **즐겨찾기** 필터로 바로 접근 가능

### 사용/미사용 설정

| 상태 | 설명 |
|------|------|
| **사용 ON** | 진료 화면 템플릿 목록에 표시 |
| **미사용 OFF** | 목록에서 숨김 (삭제하지 않고 임시로 비활성화) |

### 새 템플릿 만들기

1. 설정 → 템플릿 → **"+ 새 템플릿"** 클릭
2. **카테고리 선택** (예: 감기, 위염, 고혈압 등)
3. **템플릿 제목** 입력
4. **내용 작성** (복약 안내, 주의사항, 다음 방문 안내 등)
5. **사용 여부 체크** — 바로 사용할 경우 ON으로 설정
6. **확인** 클릭 → 저장 완료

### 기존 템플릿 수정

1. 수정할 템플릿 선택 → **미리보기**로 내용 확인
2. **"수정하기"** 버튼 클릭 → 편집 모드 활성화
3. 내용 수정 후 **"저장"** 클릭

## 병원 공용 템플릿

관리자가 설정한 병원 전체 공용 템플릿은 모든 의료진이 사용할 수 있습니다.\
공용 템플릿 관리는 관리자 권한이 필요합니다.
