# 알림 설정하기

V.Doc Pro는 환자의 **사전 문진이 완료되면 PC 메신저 알림**으로 즉시 알려드립니다.\
알림이 정상적으로 수신되려면 Windows PC 시스템 설정을 아래와 같이 확인해 주세요.

![](<../.gitbook/assets/screen-notification.png>)

---

## PC 시스템 알림 설정 확인

알림 기능이 정상 동작하려면 PC 시스템의 **방해 금지 모드가 꺼져 있어야** 합니다.

**설정 경로**: Windows 설정 → 시스템 → 알림

| 항목 | 올바른 설정 |
|------|------------|
| **알림** | 켬 |
| **방해 금지** | 꺼짐 |
| **VDoc-Pro** (앱별 알림) | 켬 |

{% hint style="warning" %}
방해 금지 모드가 **켜져 있으면** 문진 완료 알림이 수신되지 않습니다.\
Windows 설정 → 시스템 → 알림에서 방해 금지를 **꺼짐**으로 변경해 주세요.
{% endhint %}

---

## 앱별 알림 추가 설정 (선택)

VDoc-Pro 앱 알림에서 **배너**와 **소리**를 개별적으로 설정할 수 있습니다.

**설정 경로**: Windows 설정 → 시스템 → 알림 → VDoc-Pro

| 항목 | 설명 |
|------|------|
| 배너 | 화면 우측 하단에 팝업 알림 표시 |
| 소리 | 알림 수신 시 알림음 재생 |
