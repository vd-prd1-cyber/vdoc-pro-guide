# 의료진 계정 (일반관리자)

의료진 계정은 진료 화면에서 AI 문진 요약 확인, 녹음 STT, 차팅 기록 등을 사용합니다.

---

## 계정 생성 방법

관리자가 아래 순서로 진행합니다.

1. **관리자 페이지 → 계정 관리 → "+ 사용자 추가"** 클릭
2. 이름, 이메일 입력
3. 역할에서 **"의료진"** 선택
4. 담당 진료과 배정
5. **초대 이메일 발송** 클릭
6. 의료진이 이메일 링크 클릭 → 비밀번호 설정 → 로그인 완료

---

## 의료진 계정으로 할 수 있는 것

| 기능 | 설명 |
|------|------|
| **진료 화면 접근** | 전체 모드 / 미니 모드 선택 사용 |
| **AI 문진 요약 확인** | C.C., Onset, PHx, P.I. SOAP 구조 요약 확인 |
| **AI 추천 의증** | MOCA가 제시하는 감별 진단 목록 확인 및 선택 |
| **진료 녹음 STT** | 진료 대화 실시간 녹음 및 자동 요약 |
| **첨부 서류 OCR** | 환자 첨부 서류 텍스트 자동 변환 및 복사 |
| **템플릿 전송** | 진료 후 환자 맞춤 카카오톡 메시지 발송 |
| **개인 템플릿 관리** | 자주 쓰는 차팅 형식·문구 개인 저장 |

---

## 첫 로그인 후 권장 설정

1. **진료 화면 모드 선택** — 전체 모드 또는 미니 모드 중 선호하는 방식 설정
2. **개인 템플릿 등록** — 자주 쓰는 진료 후 안내 문구 미리 저장
3. **PC 알림 설정 확인** — 사전문진 완료 알림을 받으려면 PC 시스템 알림에서 **방해 금지 모드 OFF** 설정

{% hint style="info" %}
전공의는 의료진 역할로 계정을 생성하되, 필요에 따라 특정 기능 제한을 설정할 수 있습니다. 자세한 내용은 AITRICS 담당자에게 문의하세요.
{% endhint %}
