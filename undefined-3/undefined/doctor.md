# 진료화면 (의사용)

의사용 진료 화면은 AI 문진 요약 확인, 녹음 STT, AI 추천 의증, 차팅 기록까지 진료 전 과정을 하나의 화면에서 처리합니다.

***

## 진료 흐름

```
환자 카드 선택 (진료 전)
    ↓
AI 문진 요약 확인 (C.C. / Onset / PHx / P.I.)
    ↓
진료 녹음 시작 (●)
    ↓
AI 추천 의증 확인 및 선택
    ↓
진료 저장 → 템플릿 발송
```

***

## 진료 상태 구분

| 상태        | 설명                                |
| --------- | --------------------------------- |
| **진료 전**  | 환자 카드 선택 시 → 진료 중으로 전환            |
| **진료 중**  | 문진 요약, 녹음, AI 의증 활용               |
| **진료 보류** | 검사·시술이 있는 경우 보류 상태로 변경, 보류 탭으로 이동 |
| **진료 완료** | 진료 저장 클릭 또는 다른 환자 카드 선택 시 완료 처리   |

***

## 화면 모드

V.Doc Pro는 진료 환경에 맞게 두 가지 모드를 제공합니다.

| 모드                                          | 특징                    | 적합한 환경           |
| ------------------------------------------- | --------------------- | ---------------- |
| [**전체 모드**](../../menu/doctor/full-mode.md) | 모든 기능을 하나의 넓은 화면에서 사용 | 전용 모니터가 있는 진료실   |
| [**미니 모드**](../../menu/doctor/mini-mode.md) | 작은 창으로 EMR과 나란히 사용    | EMR 화면과 함께 사용할 때 |

진료 화면 우측 상단의 **"미니 모드"** 버튼으로 언제든지 전환할 수 있습니다.
