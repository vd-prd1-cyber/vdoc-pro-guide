# Cloud 설치

{% hint style="info" %}
Cloud 방식은 별도 서버 구축 없이 인터넷 연결만으로 즉시 사용할 수 있습니다. 대부분의 의원·중소병원에 권장합니다.
{% endhint %}

## 설치 절차

### STEP 1. 도입 문의 및 계약

AITRICS 담당자와 상담 후 서비스 계약을 체결합니다.

→ [서비스 신청하기](../index.md)

### STEP 2. 병원 정보 등록

AITRICS 담당자가 아래 정보를 시스템에 등록합니다.

* 병원명, 진료과 구성
* 관리자 이메일 및 연락처
* 사용 예정 계정 수 (의사, 간호사, 관리자)

### STEP 3. 커스텀 문진 설계

각 진료과 담당 의사의 요구사항에 맞게 사전문진 항목을 설계합니다.

* 진료과별 커스텀 문진 항목 선택 (단일 선택 / 복수 선택 / 숫자 입력 / 글자 입력)
* 공통 AI 문진 5세트 자동 구성
* 초진·재진 문진 흐름 확인

### STEP 4. EMR 연동 설정

| 연동 방식      | 설명                                           |
| ---------- | -------------------------------------------- |
| **API 연동** | 병원 EMR과 V.Doc Pro 자동 연동 (환자 정보, 예약 정보 자동 반영) |
| **수동 방식**  | EMR 연동 없이 V.Doc Pro 단독 운영 (환자 직접 등록)         |

{% hint style="warning" %}
연동 가능한 EMR 목록은 도입 상담 시 확인해 드립니다. 목록에 없는 EMR도 API 방식으로 연동 개발이 가능합니다.
{% endhint %}

### STEP 5. 테스트 및 교육

AITRICS CES(Customer Engineering Specialist)가 직접 방문하여 진행합니다.

* 실제 환경에서 문진·STT·템플릿 기능 테스트
* 역할별 (의사 / 간호사 / 관리자) 사용법 교육
* 마이크 설치 및 PC 알림 설정 지원
