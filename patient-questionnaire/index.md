# 사전 문진 안내 (환자용)

이 페이지는 **병원에서 사전문진 링크를 받은 환자분**을 위한 안내입니다.

---

## 사전문진이란?

진료 전날 또는 내원 당일, 카카오톡이나 문자로 받은 링크에서 증상을 미리 입력하는 것입니다.\
별도 앱 설치 없이 스마트폰 브라우저에서 바로 작성할 수 있습니다.

---

## 작성 방법

### 집에서 미리 하는 경우 (링크 문진)

1. 카카오톡 또는 문자로 받은 **사전문진 링크** 클릭
2. 본인 확인 정보 입력 (이름, 생년월일, 휴대폰 번호)
3. 필요한 경우 서류(검사 결과지, 소견서 등) 사진 첨부
4. 화면의 질문에 따라 증상 순서대로 입력 (평균 3–5분 소요)
5. **"제출"** 클릭 → 완료

### 병원에서 하는 경우 (QR 접수)

1. 접수처 앞 **QR 코드 스캔**
2. 문진 링크로 미리 하신 경우 → 본인 확인 정보 입력 후 접수 확인
3. 문진을 아직 안 하신 경우 → QR 스캔 후 현장에서 문진 시작

---

## 사용 플로우 (On-premise 환경 기준)

| 단계 | 장소 | 내용 |
|------|------|------|
| **방문 전** | 집 | 예약 확인 문자에 첨부된 교수님별 URL로 접속 → 본인 정보 확인 후 커스텀 문진·AI 문진 순차 진행. 필요 시 진료 관련 서류 첨부 업로드 가능 |
| **방문 당일** | 병원 | 접수처 앞 진료과별 QR 코드 스캔 → 휴대폰 번호로 본인 확인 후 접수 완료 |
| **문진 미진행·중단** | 병원 | QR 스캔 시 사전문진을 처음부터 시작하거나 중단 지점부터 재시작 → 완료 후 진료 대기 상태로 전환 |

완료된 사전문진 결과는 의료진의 진료 화면에서 구조화된 AI 요약(C.C.·Onset·P.I.·PHx. 등)으로 즉시 활용됩니다. 의사가 환자를 만나기 전부터 주요 증상을 파악할 수 있도록 지원합니다.

## 문진 흐름

```
본인 확인 → 서류 첨부 (선택) → 주 증상 입력 → 추가 질문 응답 → 제출 완료
```

---

## 자주 묻는 질문

**Q. 링크가 만료되었다고 나와요.**\
병원 접수 데스크에 문의해 주세요. 직원이 새 링크를 발송하거나 대신 입력해 드립니다.

**Q. 문진을 중간에 중단했어요.**\
QR 코드를 다시 스캔하면 이어서 작성하거나 처음부터 다시 시작할 수 있습니다.

**Q. 스마트폰이 없어요.**\
내원 후 접수 데스크에서 태블릿으로 작성하실 수 있습니다.

**Q. 입력한 내용이 저장되나요?**\
입력하신 내용은 담당 의료진에게만 전달되며, 개인정보보호법에 따라 암호화하여 안전하게 관리됩니다.

**Q. 어떤 질문이 나오나요?**\
주 증상, 증상 발생 시점, 과거력, 복용 약물, 알레르기 여부 등 진료과별 맞춤 질문이 나옵니다.\
AI가 증상에 따라 추가 질문을 자동으로 생성하며, 전체 11~15개 문항 이내로 구성됩니다.

**Q. 서류를 첨부해야 하나요?**\
선택 사항입니다. 다른 병원 검사 결과지나 소견서가 있다면 사진으로 첨부하시면 의사 선생님이 미리 확인할 수 있습니다.
