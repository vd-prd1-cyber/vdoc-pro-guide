# 템플릿 활성화 · 즐겨찾기 설정

진료 후 환자에게 발송할 템플릿을 검색·필터링하고, 활성/비활성 및 즐겨찾기를 설정하여 관리할 수 있습니다.

![](<../.gitbook/assets/screen-template.png>)

---

## 화면 구성

| 영역 | 설명 |
|------|------|
| **검색창** | 템플릿 제목 기준 실시간 필터링 |
| **템플릿 유형** | 전체 / 사용중인 템플릿 / 미사용 템플릿 필터 |
| **카테고리** | 카테고리별 필터링, 카테고리 추가 가능 |
| **템플릿 카드 목록** | 즐겨찾기·활성화 토글·카테고리·더보기 메뉴 포함 |
| **우측 미리보기** | 선택한 템플릿의 실제 내용 미리보기 및 수정 |

---

## 즐겨찾기 설정

템플릿 카드 좌측 상단 **★ 별 아이콘**을 클릭하면 즐겨찾기로 등록됩니다.\
즐겨찾기 등록된 템플릿은 카테고리 필터에서 **즐겨찾기** 탭으로 빠르게 조회할 수 있습니다.

{% hint style="info" %}
사용 빈도가 높은 템플릿을 즐겨찾기로 설정해 두면 진료 화면의 템플릿 선택에서 빠르게 찾을 수 있습니다.
{% endhint %}

---

## 활성화 · 비활성화

템플릿 카드 우측 상단 **토글**로 사용 여부를 설정합니다.

| 상태 | 설명 |
|------|------|
| **사용 (ON)** | 진료 화면 템플릿 목록에 표시됨 |
| **미사용 (OFF)** | 진료 화면 템플릿 목록에서 숨겨짐 |
