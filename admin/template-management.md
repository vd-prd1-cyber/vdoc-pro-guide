# 템플릿 관리

{% hint style="info" %}
관리자는 병원 전체에서 사용할 공용 진료 후 기록 템플릿을 만들고 관리합니다. 의사 개인 템플릿은 각 의사가 직접 관리합니다.
{% endhint %}

## 설정 화면 진입

**관리자 페이지 → 템플릿 관리** 탭을 클릭합니다.

## 템플릿 구조

각 템플릿은 **섹션**으로 구성됩니다. 기본 SOAP 구조에 진료과 특화 섹션을 추가할 수 있습니다.

| 섹션 유형 | 설명 |
|----------|------|
| **SOAP** | Subjective / Objective / Assessment / Plan 표준 양식 |
| **자유 텍스트** | 형식 없는 서술형 기록 |
| **구조화 필드** | 수치 입력, 체크박스, 드롭다운 등 |
| **AI 자동 채움** | 문진 요약·STT 결과 자동 삽입 섹션 |

## 새 공용 템플릿 만들기

1. **"+ 새 템플릿"** 클릭
2. 템플릿 이름 입력 (예: "상기도감염 기본", "당뇨 재진")
3. 진료과 지정 (또는 전체 공용)
4. 섹션 구성
   - 드래그 앤 드롭으로 섹션 순서 조정
   - 각 섹션의 AI 자동 채움 여부 설정
5. **저장 및 활성화**

### AI 자동 채움 매핑

섹션별로 어떤 AI 데이터를 자동으로 채울지 설정합니다.

| 섹션 | 매핑 가능한 AI 데이터 |
|-----|-------------------|
| Subjective | 문진 주 호소, AI 요약 |
| Assessment | 감별 진단 힌트 |
| Plan | STT에서 추출한 처방·교육 계획 |
| 커스텀 | 원하는 항목 선택 |

{% hint style="success" %}
AI 자동 채움 섹션은 의사가 확인 후 수정하는 방식으로 작동합니다. AI 내용이 100% 정확하지 않을 수 있으므로 최종 확인은 항상 의사가 진행합니다.
{% endhint %}

## 템플릿 관리 작업

| 작업 | 방법 |
|-----|------|
| **수정** | 템플릿 이름 클릭 → 편집 모드 |
| **복제** | ··· 메뉴 → "복사본 만들기" |
| **비활성화** | 토글 끄기 (기존 기록에는 영향 없음) |
| **삭제** | 사용 이력이 없는 템플릿만 삭제 가능 |
| **기본값 지정** | 특정 진료과의 기본 템플릿으로 설정 |

## 의사 개인 템플릿 관리

관리자는 소속 의사들의 개인 템플릿 목록을 조회할 수 있습니다 (내용은 볼 수 없고 이름·생성일만 표시).

병원 정책상 특정 개인 템플릿을 공용으로 전환하거나 비활성화해야 하는 경우, 해당 의사와 협의 후 진행하세요.

## 템플릿 사용 통계

관리자 페이지 하단에서 템플릿별 사용 빈도를 확인할 수 있습니다.

```
이번 달 가장 많이 사용된 템플릿
1위. 상기도감염 기본    584회
2위. 당뇨 재진          312회
3위. 고혈압 정기 방문   287회
```

사용 빈도가 낮은 템플릿은 통합하거나 삭제하여 목록을 정리할 수 있습니다.
