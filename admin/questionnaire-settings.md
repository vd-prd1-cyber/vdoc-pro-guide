# 문진 설정

{% hint style="info" %}
관리자는 진료과·병원 특성에 맞게 문진 항목을 커스터마이즈하고, QR 코드 발송 방식·만료 시간 등을 설정합니다.
{% endhint %}

## 설정 화면 진입

**관리자 페이지 → 문진 설정** 탭을 클릭합니다.

관리자 권한이 있는 계정으로만 접근할 수 있습니다. 권한 관리는 [계정·권한 관리](../undefined-12/account.md)를 참고하세요.

## 문진 항목 구성

### 기본 항목 (필수, 변경 불가)

* 주 증상 (자유 입력 + 키워드 선택)
* 증상 발생 시점
* 통증 강도 (NRS 0–10)
* 알레르기 여부

### 선택 항목 (표시/숨김 설정 가능)

| 항목       | 기본값 | 권장 진료과       |
| -------- | --- | ------------ |
| 현재 복용 약물 | 표시  | 내과, 심장내과     |
| 흡연·음주    | 숨김  | 호흡기내과, 소화기내과 |
| 생리 주기    | 숨김  | 산부인과         |
| 보호자 관계   | 숨김  | 소아과          |
| 해외여행 이력  | 숨김  | 감염내과         |
| 직업·환경 노출 | 숨김  | 직업환경의학과      |

항목 옆 **토글**을 클릭하여 표시 여부를 변경합니다.

### 커스텀 항목 추가

병원 고유의 문진 항목을 추가할 수 있습니다.

1. **"+ 항목 추가"** 클릭
2. 항목 유형 선택: 단답형 / 선택형 / 체크박스 / 숫자 입력
3. 질문 텍스트 입력
4. 필수 여부 설정
5. **저장**

{% hint style="warning" %}
커스텀 항목은 최대 10개까지 추가할 수 있습니다. 항목이 너무 많으면 환자 문진 완료율이 낮아질 수 있습니다.
{% endhint %}

## 진료과별 프로파일

여러 진료과가 있는 병원은 진료과별로 다른 문진 세트를 구성할 수 있습니다.

1. **"+ 프로파일 추가"** 클릭
2. 진료과 선택
3. 해당 진료과 문진 항목 구성
4. 저장

환자 접수 시 배정된 진료과에 따라 적절한 문진이 자동으로 표시됩니다.

## QR 코드 설정

| 설정 항목      | 옵션                      |
| ---------- | ----------------------- |
| **만료 시간**  | 접수 후 1시간 / 2시간 / 당일 자정  |
| **발송 채널**  | 문자(SMS) / 카카오 알림톡 / 둘 다 |
| **동의서 팝업** | 녹음 동의 팝업 활성화·비활성화       |
| **언어**     | 한국어 / 영어 / 기타 (추가 문의)   |

## 문진 완료율 모니터링

설정 화면 하단에서 최근 30일 문진 완료율 통계를 확인할 수 있습니다.

```
문진 완료율: 78.4%  ▲ 3.2% (전월 대비)
평균 소요 시간: 2분 48초
미완료 주요 이유: 스마트폰 없음 (12%) / 시간 부족 (8%)
```

완료율을 높이려면 문진 항목 수를 줄이거나, 접수 직후 즉시 QR을 발송하도록 설정하세요.
