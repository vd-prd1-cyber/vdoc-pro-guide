# 환자 목록 관리

{% hint style="info" %}
관리자와 접수 담당자는 환자 목록 화면에서 당일 예약·접수 현황을 실시간으로 관리합니다.
{% endhint %}

## 환자 목록 화면 구성

메인 화면 왼쪽에 **당일 환자 목록**이 표시됩니다.

| 컬럼 | 설명 |
|-----|------|
| **이름** | 환자 이름 (클릭 시 상세 보기) |
| **접수 시간** | 접수 완료 시각 |
| **문진 상태** | 미완료 / 완료 / AI 분석 중 / 준비 완료 |
| **진료 상태** | 대기 중 / 진료 중 / 진료 완료 |
| **담당의** | 배정된 의사 |

## 환자 추가·수정

### 새 환자 등록

1. **"+ 환자 추가"** 버튼 클릭
2. 이름, 생년월일, 연락처 입력
3. 보험 유형 선택 (건강보험 / 의료급여 / 자비)
4. **저장** 클릭

{% hint style="warning" %}
EMR 연동이 활성화된 경우 환자 정보는 EMR에서 자동으로 불러옵니다. 이 경우 V.Doc Pro에서 직접 입력할 필요가 없습니다.
{% endhint %}

### 환자 정보 수정

환자 이름을 클릭하고 **"편집"** 버튼을 누릅니다. 연락처, 보험 정보, 담당의를 변경할 수 있습니다.

### 환자 삭제·비활성화

법적 보존 의무에 따라 진료 기록이 있는 환자는 삭제되지 않습니다. 대신 **"비활성화"** 처리하여 목록에서 숨길 수 있습니다.

## 문진 상태 관리

| 상태 | 의미 | 조치 |
|-----|------|------|
| 🔴 **미완료** | 환자가 문진을 시작하지 않음 | QR 재발송 또는 대리 입력 |
| 🟡 **진행 중** | 환자가 문진 작성 중 | 대기 |
| 🔵 **분석 중** | AI 요약 생성 중 (보통 30초 이내) | 대기 |
| 🟢 **준비 완료** | 의사가 즉시 진료 가능 | 진료실로 안내 |

## 문진 QR 코드 재발송

환자가 QR 코드를 분실하거나 만료된 경우:

1. 환자 이름 옆 **점 세 개(···)** 메뉴 클릭
2. **"QR 재발송"** 선택
3. 문자 또는 카카오톡으로 재전송

## 당일 통계

목록 상단에 당일 집계가 표시됩니다.

```
총 예약: 42명  |  접수 완료: 38명  |  진료 완료: 27명  |  대기 중: 11명
```

이 수치는 1분마다 자동으로 갱신됩니다.
