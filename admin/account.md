# 계정 · 권한 관리

{% hint style="info" %}
병원 내 V.Doc Pro 사용자 계정을 만들고, 역할에 맞는 권한을 부여합니다. 권한 설정은 관리자만 변경할 수 있습니다.
{% endhint %}

## 역할 체계

V.Doc Pro는 세 가지 기본 역할을 제공합니다.

<table><thead><tr><th width="122.390625">역할</th><th width="350.7734375">주요 권한</th><th>해당 직군</th></tr></thead><tbody><tr><td><strong>관리자</strong></td><td>모든 설정 변경, 사용자 관리, 통계 조회</td><td>원장님, IT 담당자</td></tr><tr><td><strong>의사</strong></td><td>환자 진료, 문진 확인, 기록 작성, 개인 템플릿 관리</td><td>원장, 교수, 전공의</td></tr><tr><td><strong>간호사·접수</strong></td><td>환자 등록·접수, 문진 상태 확인, QR 발송</td><td>간호사, 접수 직원</td></tr></tbody></table>

전공의는 **의사** 역할로 설정하되, 필요에 따라 특정 기능(예: 처방 최종 확인)을 제한하는 커스텀 권한을 부여할 수 있습니다. 자세한 내용은 CES 담당자에게 문의하세요.
{% endhint %}

## 사용자 추가

### 단일 계정 생성

1. **관리자 페이지 → 계정 관리 → "+ 사용자 추가"** 클릭
2. 이름, 이메일, 연락처 입력
3. 역할 선택 (관리자 / 의사 / 간호사·접수)
4. 진료과 배정 (선택사항)
5. **초대 이메일 발송** 클릭

초대 이메일을 받은 사용자가 링크를 클릭하고 비밀번호를 설정하면 계정이 활성화됩니다.

### 일괄 등록 (CSV)

사용자가 많은 경우 CSV 파일로 한 번에 등록할 수 있습니다.

1. **"CSV로 일괄 등록"** 클릭
2. 제공된 템플릿 CSV 다운로드
3. 사용자 정보 입력 후 업로드
4. 오류 검토 후 **"등록 확인"** 클릭

CSV 형식: `이름, 이메일, 역할, 진료과` (첫 줄은 헤더)

## 계정 관리

| 작업           | 방법                           |
| ------------ | ---------------------------- |
| **비밀번호 초기화** | 계정 옆 ··· → "비밀번호 재설정 이메일 발송" |
| **역할 변경**    | 계정 클릭 → 역할 드롭다운 변경 → 저장      |
| **계정 잠금**    | 퇴직·휴직 시 토글 끄기 (데이터는 보존)      |
| **계정 삭제**    | 잠금 후 30일 경과 시 삭제 가능          |

{% hint style="success" %}
**계정 잠금 vs 삭제** 퇴직한 직원 계정은 즉시 삭제하지 말고 **잠금** 처리를 권장합니다. 해당 직원이 작성한 진료 기록의 작성자 정보가 유지되어야 하기 때문입니다.
{% endhint %}

## 로그인 보안 설정

| 설정           | 옵션                         |
| ------------ | -------------------------- |
| **세션 만료**    | 30분 / 1시간 / 4시간 / 8시간 / 없음 |
| **2단계 인증**   | 선택 / 필수 (관리자 계정 권장)        |
| **IP 허용 목록** | 병원 내부 IP만 허용 (선택사항)        |
| **비밀번호 정책**  | 8자 이상 / 90일마다 변경 등         |

## 접속 이력 조회

보안 감사 목적으로 사용자별 접속 이력을 확인할 수 있습니다.

**관리자 페이지 → 보안 → 접속 이력**

| 항목    | 표시 내용           |
| ----- | --------------- |
| 접속 일시 | 로그인·로그아웃 시각     |
| 접속 IP | 접속 위치           |
| 기기 정보 | 브라우저·OS 정보      |
| 이상 징후 | 비정상 접속 시 자동 플래그 |

이력은 최대 1년치를 보관합니다.
