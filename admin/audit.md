# 감사 기록

{% hint style="info" %}
감사 기록은 **관리자 계정**에서만 조회할 수 있습니다.
{% endhint %}

V.Doc Pro 내에서 발생한 주요 활동 이력을 기록합니다.\
보안 감사, 개인정보 처리 현황 확인, 이상 접근 탐지에 활용됩니다.

---

## 조회 방법

**관리자 페이지 → 감사 기록** 탭 클릭

---

## 기록 항목

| 항목 | 내용 |
|------|------|
| **사용자 접속 이력** | 로그인·로그아웃 일시, 접속 IP, 기기·브라우저 정보 |
| **계정 변경 이력** | 계정 생성, 역할 변경, 잠금·해제 처리 내역 |
| **설정 변경 이력** | 문진 항목, 템플릿, 보안 설정 변경 내역 |
| **데이터 접근 이력** | 환자 정보 조회, 문진 결과 열람 기록 |
| **이상 징후 플래그** | 비정상 접속 패턴 자동 감지 시 표시 |

---

## 보관 기간 및 검색

- 이력 데이터는 최대 **1년치** 보관
- 기간, 사용자, 활동 유형별 필터 검색 지원
- 특정 이벤트 로그는 CSV로 내보내기 가능

---

## 활용 방법

| 목적 | 활용 방법 |
|------|---------|
| **보안 감사** | 정기적으로 접속 이력 확인, 이상 접근 탐지 |
| **퇴직자 관리** | 계정 잠금 후 해당 계정의 마지막 접속 이력 확인 |
| **개인정보 처리 증빙** | 데이터 접근 이력 내보내기 (감사 자료 제출 시) |
| **이슈 추적** | 문제 발생 시점의 시스템 활동 역추적 |
