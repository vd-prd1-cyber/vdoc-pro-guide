# 오늘 현황 보기

당일 진료 환자 현황을 의사/진료실 단위 컬럼 보드로 한눈에 확인할 수 있습니다.\
각 환자의 진료 상태와 문진 완료 여부, 템플릿 발송 여부를 빠르게 파악할 수 있습니다.

![](<../.gitbook/assets/screen-today.png>)

---

## 화면 구성

**의사별 컬럼**\
화면 상단에 의사(교수님)별로 컬럼이 나뉘어 표시됩니다.\
각 컬럼 상단의 이름을 클릭하면 해당 의사의 진료 현황으로 이동할 수 있습니다.

**상태 탭**\
각 컬럼 상단에는 **진료 · 보류 · 완료** 3개 탭이 있으며, 탭 우측에 해당 상태의 환자 건수가 표시됩니다.\
탭을 클릭하면 해당 상태의 환자 카드만 표시됩니다.

| 탭 | 의미 |
|----|------|
| **진료** | 현재 진료 진행 중 또는 진료 직전/직후 관리가 필요한 환자 |
| **보류** | 진료 보류 상태 또는 후속 확인이 필요한 환자 |
| **완료** | 진료 완료 또는 해당 세션 종료 상태 환자 |

---

## 환자 카드

각 환자 카드에는 다음 정보가 표시됩니다.

| 항목 | 내용 |
|------|------|
| 환자명 · 번호 · 성별/나이 | 카드 상단에 표시 |
| 접수 시간 | 카드 본문에 표시 (예: 오전 09:03) |
| 상태칩 | 현재 상태 표시 (문진완료, 문진생략, 문진중, 도착 등) |
| X 아이콘 | 클릭 시 해당 환자 카드를 보드에서 제거 |

카드를 클릭하면 해당 환자의 진료 조회 화면으로 이동합니다.

{% hint style="info" %}
카드는 접수 시간순으로 정렬됩니다.\
X 아이콘으로 카드를 닫으면 목록에서 즉시 제거됩니다.
{% endhint %}

---

## 템플릿 발송

**완료** 탭의 환자 카드 하단에 **템플릿 발송** 버튼이 표시됩니다.\
버튼을 클릭하면 템플릿을 선택하여 해당 환자에게 발송할 수 있습니다.

---

## 환자 카드 상태

**접수 취소**\
환자 카드 우측 상단 **X 버튼**을 누르면 접수 취소 안내창이 표시됩니다.\
**확인**을 클릭하면 접수가 취소되고 현황 목록에서 즉시 삭제됩니다.

**템플릿 미발송**\
완료 탭 환자 카드에 **템플릿 발송** 버튼이 표시되면 아직 템플릿이 발송되지 않은 상태입니다.\
버튼을 클릭하여 템플릿을 선택하고 발송할 수 있으며, 과거 날짜의 진료 건도 발송 가능합니다.

**템플릿 발송 완료**\
템플릿 발송이 완료된 환자 카드에는 발송 버튼이 표시되지 않습니다.
