# 진료 관리 화면 (간호사용)

간호사·어시스턴트용 화면은 환자 접수부터 진료 흐름 모니터링, 템플릿 발송 확인까지 진료실 전체를 관리합니다.

---

## 주요 메뉴

| 메뉴 | 기능 |
|------|------|
| **[오늘 현황](today.md)** | 의사별 당일 진료 현황, 환자 카드 상태 실시간 확인 |
| **[환자 관리](patients.md)** | 일단위·주단위 진료실 현황, 환자 카드 상태 변경 |
| **[환자 목록](list.md)** | 진료 완료 환자 전체 목록, 이력 조회 |

---

## 간호사 업무 흐름

```
예약 확인 (오늘 현황)
    ↓
문진 링크 발송 (카카오톡 / SMS)
    ↓
환자 내원 → QR 체크인 확인
    ↓
문진 완료 상태 모니터링
    ↓
진료 완료 후 템플릿 발송 확인
```
